# 司马迁

司马迁把庞杂历史组织为彼此照应的人物行动。他既记录制度、战争和权力，也关注失败者、边缘人物与个人选择，使历史叙述具有持续的情感张力和伦理复杂性。

## 文学史位置

《史记》的本纪、世家、列传并非资料的简单分类，而是一套通过人物关系理解历史的叙事结构。后世传记、小说和戏剧都从中吸收了人物塑造与命运书写的方法。
