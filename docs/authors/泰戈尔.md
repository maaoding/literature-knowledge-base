# 泰戈尔

把孟加拉传统、个人抒情与世界主义结合，并持续反思民族主义的边界。

## 文学史位置

他的作品尊重民族文化，却拒绝以排他身份代替人的自由和具体关系。

## 代表作品

- 《吉檀迦利》

## 阅读提示

诗歌之外可结合小说阅读，观察抽象人道理想在家庭和政治冲突中如何接受检验。
