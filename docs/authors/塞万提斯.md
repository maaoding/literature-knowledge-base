# 塞万提斯

塞万提斯以《堂吉诃德》成为现代小说史上的关键作家。他通过讽刺骑士小说，让文学开始反思阅读、幻想和现实之间的关系。

## 为什么读

- 理解现代小说的自反意识。
- 观察理想主义与现实秩序的冲突。
- 进入欧洲小说传统的重要源头。
