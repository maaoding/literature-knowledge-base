# 果戈里

果戈里让鼻子、外套、官阶和文书等日常物件获得不成比例的力量。人物被制度和欲望推向滑稽变形，现实因此显露出比幻想更不可信的一面。

## 文学史位置

他的写作连接浪漫主义怪诞与俄国现实主义，对陀思妥耶夫斯基、卡夫卡式官僚荒诞和现代黑色幽默都有深远影响。

## 代表作品

- [《死魂灵》](/works/死魂灵)

## 阅读提示

关注叙述者如何在讽刺、抒情和跑题之间摇摆。小说批判的并不只是几个贪婪人物，而是一整套把人变成数字和身份标签的社会语言。
