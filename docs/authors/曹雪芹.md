# 曹雪芹

曹雪芹以《红楼梦》进入文学史。它不是单纯的家族故事，而是把人物心理、日常生活、诗词才情和制度压力交织在一起的复杂小说。

## 为什么读

- 理解中国古典长篇小说的成熟形态。
- 学习人物群像和细节叙事。
- 观察情感、家族和制度之间的关系。
