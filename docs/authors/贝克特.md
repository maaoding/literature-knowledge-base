# 贝克特

通过极简舞台、重复动作和语言崩解呈现战后人的存在困境。

## 文学史位置

作品减少背景和情节，不是为了故作玄虚，而是让等待、依赖和时间本身成为可见结构。

## 代表作品

- [《等待戈多》](/works/等待戈多)

## 阅读提示

把重复当作变化来读：每次相似对话都暴露人物记忆和关系的新裂缝。
