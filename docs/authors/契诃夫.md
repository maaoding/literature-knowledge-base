# 契诃夫

契诃夫很少依靠突出的情节裁决人物。他让闲谈、误解、沉默和未完成的愿望留在作品表面，使生活中的迟滞本身成为戏剧力量。

## 文学史位置

他的短篇小说削弱传统结局，戏剧则把冲突移入潜台词和群体节奏。二十世纪小说、舞台表演和影视叙事都继承了这种开放结构。

## 代表作品

- [《樱桃园》](/works/樱桃园)

## 阅读提示

关注人物说了什么，也关注他们回避了什么。作品的转折往往发生在琐碎动作和话题错位之间，而不是戏剧性宣告中。
