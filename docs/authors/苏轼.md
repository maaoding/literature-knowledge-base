# 苏轼

苏轼是宋代文学中跨文体能力极强的作家。他的诗、词、文都能把个人遭遇、日常经验、历史感和人生哲学融合起来。

## 历史位置

苏轼扩大了词的表达范围，使词不只适合写柔婉情绪，也能写历史、人生、旷达和自我反省。他的散文则常在清晰自然的语言中展开思想。

## 阅读重点

- 旷达不是简单乐观，而是经历挫折后的自我安顿。
- 宋词中的“我”更日常、更反省，也更重视心境。
- 诗、词、文可以互相参照阅读。

## 代表作品

- 《念奴娇·赤壁怀古》
- 《赤壁赋》
