# 乔治·艾略特

乔治·艾略特把小城中的婚姻、职业、财产和理想组织成彼此牵动的社会网络。她既理解人物的自我辩护，也持续追问个人选择给他人带来的后果。

## 文学史位置

她把英国现实主义从风俗描写推进到伦理心理和复杂因果。叙述者的同情并不取消判断，而是要求读者在判断之前先理解人物所处的限制。

## 代表作品

- [《米德尔马契》](/works/米德尔马契)

## 阅读提示

不要只追随单一主人公。记录不同人物的目标如何互相阻碍或意外支持，才能看见小说真正的主角是整个小城的关系结构。
