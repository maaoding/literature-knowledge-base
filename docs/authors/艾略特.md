# T. S. 艾略特

艾略特把不同语言、时代和文类的声音压缩在同一首诗中。破碎并非单纯的形式炫技，而是现代经验失去共同秩序之后的语言状态。

## 文学史位置

他的诗歌和批评共同塑造了英语现代主义的阅读方式：传统不再是固定遗产，而是在新作品中被重新排列的关系网络。

## 阅读提示

第一次阅读不必查清每条引文。先辨认声音切换、场景重复和水意象，再借注释理解这些碎片如何形成整体压力。
