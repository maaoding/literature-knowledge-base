# 彼特拉克

以十四行诗发展出持续观察欲望、时间和自我分裂的近代抒情声音。

## 文学史位置

诗中的爱既是现实情感，也是诗人不断审视自身矛盾的镜面。

## 代表作品

- 《歌集》

## 阅读提示

按组阅读同一意象的反复变化，比孤立摘取一首诗更容易看见自我如何被时间改写。
