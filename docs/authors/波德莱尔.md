# 波德莱尔

波德莱尔把拥挤街道、橱窗、疲惫身体和转瞬即逝的陌生人带入诗歌。他不回避丑恶材料，而是追问现代城市如何改变感知、欲望和美的条件。

## 文学史位置

他的写作位于浪漫主义之后、象征主义和现代主义之前。《恶之花》的组诗结构、感官对应和都市观察，为后来的现代诗提供了重要语言模型。

## 代表作品

- [《恶之花》](/works/恶之花)

## 阅读提示

不要只寻找颓废意象。可以按组诗顺序观察“理想”与“忧郁”如何反复拉扯，以及城市中的短暂相遇如何成为新的抒情场景。
