# 梅尔维尔

梅尔维尔把航海经验、博物知识、宗教语言和戏剧独白并置在小说中。他笔下的海洋既是劳动现场，也是无法被彻底解释的认识边界。

## 文学史位置

他的写作跨越浪漫主义象征、现实观察和现代小说实验。《白鲸》在结构上的混杂与对解释欲的怀疑，使它不断被后来的现代主义重新发现。

## 代表作品

- [《白鲸》](/works/白鲸)

## 阅读提示

不要只追踪捕鲸情节。小说中的分类、讲演和旁逸斜出共同构成主题：人如何试图把不可穷尽的世界变成可控制的知识。
