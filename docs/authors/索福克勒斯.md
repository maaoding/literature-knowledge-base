# 索福克勒斯

以结构严密的悲剧检验知识、责任和城邦法律之间难以调和的冲突。

## 文学史位置

人物往往因坚持追问真相而走向灾难，悲剧不否定行动，却揭示行动者无法掌握全部条件。

## 代表作品

- [《俄狄浦斯王》](/works/俄狄浦斯王)

## 阅读提示

关注观众知道、人物不知道的信息差，以及真相如何一步步由拯救希望变成毁灭力量。
