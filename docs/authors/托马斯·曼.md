# 托马斯·曼

托马斯·曼用表面从容、内里反讽的叙述处理家庭衰败、艺术家身份和欧洲观念危机。他的人物常被放入封闭空间，让时代矛盾以日常谈话和身体经验显现。

## 文学史位置

他的小说保留现实主义的社会观察和人物连续性，同时吸收现代主义对时间、疾病与不可靠价值的关注，形成连接十九世纪和二十世纪的重要桥梁。
