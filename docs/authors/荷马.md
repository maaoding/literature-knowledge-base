# 荷马

荷马史诗建立了西方叙事关于英雄行动、战争共同体和归乡考验的基本范式。

## 文学史位置

史诗声音兼顾个人荣誉与集体命运，通过反复场景、固定修饰语和大规模比喻维持口头传统的力量。

## 代表作品

- [《伊利亚特》](/works/伊利亚特)
- [《奥德赛》](/works/奥德赛)

## 阅读提示

先分清《伊利亚特》的愤怒结构与《奥德赛》的归乡结构，再比较两种英雄观。
