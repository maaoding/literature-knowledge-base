# 易卜生

把婚姻、法律和主体自由压缩进日常客厅，建立现代社会问题剧。

## 文学史位置

家庭空间看似私人，实际由财产、名誉和法律塑造；人物冲突因此具有制度维度。

## 代表作品

- [《玩偶之家》](/works/玩偶之家)

## 阅读提示

关注秘密被揭开前后的语言变化，以及结尾行动为何比一场夫妻争吵更深远。
