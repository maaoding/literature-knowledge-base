# 鲁迅

鲁迅是现代中国文学的关键入口。他的小说常以冷峻的叙事呈现旧秩序中的个体困境，杂文则把思想批评、语言锋芒和公共讨论结合起来。

## 为什么读

- 进入现代中国文学的问题意识。
- 理解现实主义、启蒙和现代性之间的张力。
- 学会从短篇小说观察人物、社会结构和叙事角度。
