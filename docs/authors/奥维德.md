# 奥维德

奥维德把分散的希腊罗马神话串联为不断变化的叙事链。人物、植物、动物和星辰彼此转化，故事也在讲述者之间连续转场，使古代神话获得高度自觉的文学结构。

## 文学史位置

他的神话重述深刻影响欧洲绘画、戏剧和诗歌。作品既保存古典故事，又以机智、距离感和叙事速度重新解释权力、欲望与身体。
