# 普希金

普希金把欧洲浪漫主义的自由形式转化为具有俄国语感的诗歌、叙事和戏剧。他的语言兼具口语速度与古典控制，使人物能够在社会习俗、个人欲望和历史力量之间自然行动。

## 文学史位置

他既是俄国浪漫主义的核心人物，也是果戈里、托尔斯泰和陀思妥耶夫斯基等后来小说家的重要起点。其作品让诗性想象逐渐进入日常社会观察。
