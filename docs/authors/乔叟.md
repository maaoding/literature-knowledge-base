# 乔叟

让不同阶层共享讲述空间，以多声部故事呈现中古晚期社会的复杂面貌。

## 文学史位置

朝圣框架暂时把社会等级不同的人聚在一起，每个故事既暴露讲述者，也打开一种文类传统。
