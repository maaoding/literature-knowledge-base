# 拉伯雷

以巨人、饮食和夸张笑声冲击禁欲规范，释放文艺复兴的世俗生命力。

## 文学史位置

作品把身体的开放、语言的过剩与知识兴趣结合，粗俗和博学并不互相排斥。

## 代表作品

- [《巨人传》](/works/巨人传)

## 阅读提示

接受它的杂乱和过度，把笑声看成对僵化权威的认识方式。
