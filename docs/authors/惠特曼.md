# 惠特曼

惠特曼用延展长句、目录式列举和直接呼告，把身体、劳动、城市、人群与自然纳入同一个诗歌声音。他的“自我”不是封闭个体，而是不断与他人交换位置的公共主体。

## 文学史位置

《草叶集》持续修订数十年，改变了英语诗歌对行长、节奏和题材的理解。现代自由诗、城市诗和公共诗歌都能在其中找到重要源头。

## 阅读提示

先朗读较短篇章，感受重复、并列和呼吸节奏。面对长篇列举时，不必逐项解释，而要观察声音如何借累积形成包容世界的姿态。
