# 鲁米

用爱情、音乐和故事表现灵魂超越狭窄自我、接近整体存在的过程。

## 文学史位置

世俗爱人、酒和舞蹈在诗中具有多重含义，感官经验并不与精神追求对立。

## 代表作品

- 《玛斯纳维》
