# 加缪

以清晰而冷峻的语言讨论世界无意义时，人仍如何行动并与他人共同生活。

## 文学史位置

荒诞不是虚无主义结论，而是人追求意义与世界沉默之间的持续张力。

## 代表作品

- [《局外人》](/works/局外人)

## 阅读提示

区分主人公的感觉、社会审判和作者的伦理问题，不要只用“冷漠”概括作品。
