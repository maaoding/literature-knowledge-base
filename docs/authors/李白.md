# 李白

李白是盛唐诗歌中最能体现想象力、自由气势和个体精神的诗人之一。他的诗常以夸张、跳跃、神话感和豪放节奏，把现实经验推向更高的精神姿态。

## 历史位置

读李白，可以看到唐诗不只是严整的格律和现实记录，也有强烈的浪漫想象、游仙传统和自我意识。他和杜甫常被并置阅读：一个偏向奔放的自我和想象，一个偏向历史经验和伦理关怀。

## 阅读重点

- 想象力：山川、星月、神仙和远游常被写成开阔空间。
- 声调和气势：诗句常有强烈推进感。
- 个体姿态：自信、孤独、饮酒、远行构成诗人形象。

## 代表作品

- 《将进酒》
- 《蜀道难》
