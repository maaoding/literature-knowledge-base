# 李白

李白的“豪放”并不只是声音响亮。他善于把山川、饮酒、离别、求仕与神仙想象压进快速变化的诗歌空间，使现实受挫与精神伸展同时存在。后世塑造的“诗仙”形象强化了天才、自由和醉酒，却容易遮蔽他对仕进的持续期待、古典传统的深厚训练，以及作品中细密的孤独与时间感。

## 生平与时代

李白生于 701 年，出生地说法不一，童年主要在蜀地度过。他没有走常规科举道路，青年后长期远游，通过交游和献诗寻求政治机会。742 年奉召入长安供奉翰林，时间不长便离开宫廷。安史之乱期间卷入永王李璘事件，被判流放夜郎，途中遇赦，晚年继续漂泊，762 年去世。传奇不断附着于其生平，阅读时应区分可靠史料、诗中自我塑造和后世逸闻。

## 文学史位置

李白综合《楚辞》、汉魏乐府、游仙诗和六朝山水传统，尤其擅长古体、歌行与绝句。他让诗中的“我”拥有强烈表演性：时而直上天界，时而对月独酌，时而在送别中突然缩回孤身。与[杜甫](/authors/杜甫)并置有助于看见盛唐诗歌的不同可能，但把二人固定为“浪漫／现实”会过度简化；李白同样回应政治与战争，杜甫也拥有奇崛想象。

## 核心主题

远游与求仕构成李白诗中的基本矛盾。他渴望建功和被理解，又不断以归隐、饮酒和仙游抵抗现实限制。友谊与送别因此特别重要：空间距离被长江、明月和孤帆拉开，也通过想象重新连接。时间意识藏在豪情内部，盛宴、青春与功名越被夸张，消逝感越明显。山水并非静止风景，而常成为身体移动、情绪转折和自我放大的媒介。

## 语言与文体

李白善用夸张、跳跃、重复和口语化呼告。长篇歌行常由短促句、散文化句式与整齐节奏交替推动，像即兴喷涌，实际依靠精密的声音控制。绝句则能删去因果，只保留一个视线移动或空间余韵。他频繁借神话、历史人物和天体扩大尺度，同时用清楚动词保持画面可感。读诗不能只摘取名句，还要追踪整篇速度何时加快、何时骤停。

## 代表作品

- 《蜀道难》：以重复呼告、神话和险峻空间制造行进压力。
- 《将进酒》：在宴饮豪情中同时书写时间、价值与失意。
- 《梦游天姥吟留别》：把梦境、山水和人格姿态连成剧烈转换。
- 《黄鹤楼送孟浩然之广陵》《静夜思》：在极短篇幅中处理视线、距离与记忆。

## 影响与接受

李白在唐代已享盛名，后世不断通过选本、绘画、戏曲和传说再造其形象。他对汉语诗歌的影响既在题材，也在一种敢于改变尺度和语气的许可。近现代翻译使李白进入英语等语言的现代诗谱系，但转译常突出清澄意象而弱化格律、典故和历史语境。“斗酒诗百篇”式大众记忆亲切有力，仍需与文本中的技艺和矛盾互相校正。

## 阅读建议

先从绝句进入，画出诗中视线和人物位置；再朗读《将进酒》《蜀道难》等歌行，标记重复、换韵与句长变化。选择一首与杜甫同题材作品对读，不急于判定谁更“真实”。最后回到[唐宋文学](/history/唐宋文学)和[诗歌传统入门](/paths/诗歌传统入门)，比较李白如何改写《楚辞》、乐府与游仙传统。版本中个别作品归属有争议，深读时可参考可靠校注本的辨伪说明。
