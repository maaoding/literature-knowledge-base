# 李清照

李清照善于从帘、花、酒、雁和梧桐等日常意象中提炼时间感。她的词既有早期生活中的明快与幽微，也有南渡之后对离乱、丧失和记忆的重新组织。

## 文学史位置

她使词体的声音、节奏和个人经验高度结合，也通过词论表达鲜明的文体意识。她的写作不能只被归结为柔婉，其中包含判断、讽刺和历史创伤。
