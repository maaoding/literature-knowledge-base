# 维吉尔

把希腊史诗形式转化为罗马建国叙事，并持续追问公共使命要求个人付出什么。

## 文学史位置

他的史诗一面为罗马秩序提供起源，一面保存战争、离乡和被牺牲者的哀伤。

## 代表作品

- [《埃涅阿斯纪》](/works/埃涅阿斯纪)

## 阅读提示

不要只把作品当帝国颂歌，应同时观察使命叙事中不断出现的损失与迟疑。
