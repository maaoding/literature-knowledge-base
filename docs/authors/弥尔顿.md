# 弥尔顿

在革命失败与宗教信念中重写创世和堕落，把自由意志置于史诗中心。

## 文学史位置

作品的宏大秩序与撒旦的修辞魅力形成危险张力，让读者不得不辨认语言和道德的距离。

## 代表作品

- [《失乐园》](/works/失乐园)

## 阅读提示

比较叙述者立场、人物言说和行动后果，不要因雄辩自动认同说话者。
