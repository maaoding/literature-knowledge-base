# 夏目漱石

以讽刺和心理分析书写明治现代化中的知识分子、城市关系与个人孤独。

## 文学史位置

现代自我追求独立，却不断在金钱、友情和伦理债务中感到不安。

## 代表作品

- 《心》

## 阅读提示

从叙述视角和沉默处入手，不要把人物的自我解释直接当成作品结论。
