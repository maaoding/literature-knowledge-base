# 庄子

庄子及其后学留下的文本把抽象思辨放进大鹏、蝴蝶、庖丁和混沌等形象之中。它不急于建立唯一答案，而是通过尺度转换、语言反讽和梦境结构，让读者重新观察知识、自由与生命边界。

## 文学史位置

《庄子》既属于先秦思想史，也深刻影响中国寓言、散文和诗歌想象。它证明说理文字可以依靠叙事、声音和形象获得文学强度。

## 阅读提示
