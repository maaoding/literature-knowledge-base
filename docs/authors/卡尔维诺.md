# 卡尔维诺

卡尔维诺不断改变小说的组织规则，却保持语言的清晰和游戏感。他让寓言、科学概念、旅行见闻和读者行为成为结构材料，使形式实验不必等同于晦涩。

## 文学史位置

他的写作连接战后现实、结构主义和后现代叙事。作品经常公开展示自身规则，让读者同时进入故事并观察故事如何被制造。

## 代表作品

- [《看不见的城市》](/works/看不见的城市)

## 阅读提示

不必寻找唯一的城市原型。可以记录每座城市围绕记忆、欲望、符号或死亡形成的规则，再观察全书如何把碎片排列成对称结构。
