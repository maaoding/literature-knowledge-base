# 海明威

海明威常把人物放入战争、狩猎、竞技和劳动等高压情境，却很少直接解释其情绪。短句、动作和沉默共同构成意义，让未说出的创伤持续留在文本下方。

## 文学史位置

他的节制文体影响了现代小说、新闻写作和影视对白。所谓“冰山”并不是内容贫乏，而是要求可见细节承担更大的经验重量。

## 代表作品

- [《老人与海》](/works/老人与海)

## 阅读提示

注意重复动作、物件和身体感觉。人物的价值不由议论直接宣布，而是在失败无法避免时如何继续行动中显现。
