# 歌德

横跨激情反叛、古典平衡和世界文学视野，是理解德国近代文学的核心作家。

## 文学史位置

他不断修正年轻时代的无限激情，试图让行动、知识和形式获得更成熟的尺度。

## 代表作品

- [《少年维特之烦恼》](/works/少年维特之烦恼)
- [《浮士德》](/works/浮士德)

## 阅读提示

把《维特》和《浮士德》对读，可看到主观激情如何转化为更复杂的现代行动问题。
