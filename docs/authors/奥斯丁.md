# 简·奥斯丁

简·奥斯丁的小说常被误读为轻巧的爱情故事。实际上，她写的是婚姻、财产、阶层、判断力和女性处境之间的精密关系。

## 历史位置

奥斯丁的长处在于把社会结构压进日常对话和家庭场景。她不依靠宏大战争或戏剧化灾难，而是在礼节、拜访、舞会、误解和婚姻选择中呈现社会规则。

## 阅读重点

- 婚姻选择背后是财产、继承和阶层秩序。
- 讽刺常藏在克制的叙述语气中。
- 女主人公的成长不是“找到对象”，而是校正判断力。
