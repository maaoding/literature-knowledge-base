# 但丁

但丁以《神曲》组织了一次从地狱到天堂的精神旅程。作品同时容纳宗教想象、政治判断、伦理秩序和个人命运。

## 为什么读

- 理解欧洲文学中的伦理旅程。
- 观察宏大结构如何组织人物、象征和知识。
- 进入中世纪与文艺复兴之间的重要文学转折。
