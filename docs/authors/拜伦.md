# 拜伦

把个人传奇、政治反抗和讽刺语言结合，塑造骄傲而孤独的现代英雄。

## 文学史位置

他的主人公反抗社会又无法摆脱自我，英雄形象始终带有戏仿和自我拆解。

## 代表作品

- 《唐璜》

不要把作者生平直接等同于作品人物，尤其要注意叙述者的讽刺距离。
