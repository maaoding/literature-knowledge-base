# 菲尔多西

以宏大民族史诗保存伊朗神话、王朝记忆和新波斯语的文学尊严。

## 文学史位置

作品把神话时代、英雄时代和历史时代连接起来，英雄荣耀始终伴随权力循环和家族悲剧。

## 代表作品

- [《列王纪》](/works/列王纪)

## 阅读提示

可从鲁斯塔姆故事进入，观察个人力量为何无法解除历史和亲族关系的悲剧。
