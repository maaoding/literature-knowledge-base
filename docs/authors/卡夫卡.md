# 卡夫卡

卡夫卡是理解现代主义和异化经验的关键作家。他的小说常用冷静、简洁甚至近乎公文式的语气，呈现荒诞而无法逃脱的处境。

## 历史位置

卡夫卡的文学世界中，问题往往没有明确起因，也没有清楚解决方案。人物面对家庭、制度、法律和身体变化时，感到自己失去解释世界的能力。这种经验构成二十世纪文学的重要现代感。

## 阅读重点

- 荒诞不是怪奇效果，而是现代处境的结构。
- 叙述语气越冷静，处境越压迫。
- 家庭、工作和制度常共同构成无形压力。
