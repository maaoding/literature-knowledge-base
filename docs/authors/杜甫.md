# 杜甫

杜甫的诗把个人命运和时代变局紧密结合，后世常称其诗为“诗史”。读杜甫，可以理解诗歌如何承担现实关怀，同时保持形式上的高度凝练。

## 为什么读

- 从诗歌进入唐代社会和士人精神。
- 理解格律、叙事和情感力度。
- 建立古典诗歌阅读的基本敏感。

## 代表作品

- 《春望》
- 《登高》
