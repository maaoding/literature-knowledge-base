# 迦梨陀娑

以精致诗剧和抒情诗把爱情、记忆与自然感受组织为古典梵语文学的高峰。

## 文学史位置

他的作品重视情感逐步成熟的过程，人物并不靠强烈行动取胜，而在误解、分离和重认中显出内在变化。

## 代表作品

- [《沙恭达罗》](/works/沙恭达罗)

## 阅读提示

阅读时留意季节、植物和空间怎样参与人物情感，而不只是充当背景。
