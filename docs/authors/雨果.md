# 雨果

以宏阔想象、强烈对照和公共激情推动法国浪漫主义进入小说与戏剧。

## 文学史位置

他把建筑、群众和被排斥者置于历史舞台，善恶对照鲜明，却不断质疑制度正义。

## 代表作品

- [《巴黎圣母院》](/works/巴黎圣母院)

## 阅读提示

关注城市空间如何保存历史，以及人物身体如何承受社会偏见。
