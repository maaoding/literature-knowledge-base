# 屈原

屈原是中国文学中强烈个人抒情传统的重要源头。相比《诗经》中更偏共同体、礼乐和民歌的声音，《楚辞》呈现出更鲜明的“我”：理想、痛苦、忠诚、放逐和神话想象被组织成高度个人化的表达。

## 历史位置

屈原的重要性在于，他把政治人格和诗性想象结合起来。后世中国文学中常见的“士人不遇”“忠而被逐”“以香草美人寄托理想”等表达，都能在楚辞传统中找到源头。

## 阅读重点

- 看“我”的出现：抒情主体比《诗经》更强。
- 看意象系统：香草、美人、神灵、远游组成象征世界。
- 看政治伦理：个人理想和现实权力之间的冲突。
