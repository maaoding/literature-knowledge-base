# 川端康成

把传统审美、感官片段和现代人的距离感结合，形成冷静而含蓄的小说。

## 文学史位置

景物和身体细节承担大量未说出的情感，人物关系常停留在无法完全抵达的距离。

## 代表作品

- [《雪国》](/works/雪国)

## 阅读提示

关注镜像、声音和季节变化，避免把含蓄叙事简化为单一爱情故事。
