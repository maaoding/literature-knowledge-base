# 莎士比亚

莎士比亚是进入世界戏剧传统的重要入口。他的作品通过人物语言、舞台冲突和复杂情境处理权力、欲望、犹疑和命运。

## 为什么读

- 理解戏剧如何通过冲突组织人物。
- 进入文艺复兴时期的人性讨论。
- 学习独白、对白和舞台结构。
