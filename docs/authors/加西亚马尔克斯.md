# 加西亚·马尔克斯

加西亚·马尔克斯把拉丁美洲历史、家族记忆和神话感放进现代小说结构中。《百年孤独》是理解魔幻现实主义的重要入口。

## 为什么读

- 理解魔幻现实主义如何处理历史记忆。
- 观察家族叙事和循环结构。
- 扩展现代小说的世界文学视野。
