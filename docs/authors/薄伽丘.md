# 薄伽丘

薄伽丘让一群躲避瘟疫的青年以讲故事维持秩序。不同身份、语气和价值判断在框架中并存，使日常欲望、交易、机智和宗教虚伪成为可观察的社会生活。

## 文学史位置

《十日谈》继承民间故事、传奇和说笑传统，又以城市经验与人的行动能力打开文艺复兴叙事。它也是欧洲短篇小说和框架故事的重要源头。

## 阅读提示

不要把故事只当作趣闻。比较不同讲述者如何处理相似主题，并观察灾难背景与欢快叙述之间始终存在的张力。
