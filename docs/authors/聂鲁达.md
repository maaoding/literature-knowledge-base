# 聂鲁达

从私人爱情扩展到自然、劳动者和拉丁美洲历史，形成跨度巨大的诗歌声音。

## 文学史位置

他的诗既有物质感和感官密度，也试图承担政治见证，二者之间始终存在张力。

## 代表作品

- 《漫歌》

## 阅读提示

分阶段阅读爱情诗和历史诗，比较“我”的尺度如何从个人扩展为大陆。
