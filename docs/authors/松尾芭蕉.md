# 松尾芭蕉

把俳句从机智游戏推进为凝练的旅行观察和生命体悟。

## 文学史位置

极短诗形通过季语和留白连接眼前景象、文化记忆与行旅者的身体感。

## 代表作品

- 《奥州小道》

## 阅读提示

不要只寻找哲理结论，先感受景物之间的距离、声音和停顿。
