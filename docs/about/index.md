# 编辑说明

文学知识库面向希望建立阅读坐标的中文读者。它不是作品榜单，也不试图替代专业文学史、校勘本或学术论文，而是把文学史、作家、作品、专题、理论与技巧组织成可继续追索的阅读网络。

## 选目原则

站点优先选择能够呈现重要文学传统、文体变化或批评问题的作家与作品。入选不等于价值排名；篇幅、国别和时代分布也不代表完整的世界文学版图。现有缺口会按文学史关联、来源条件和读者可进入性逐步补充。

## 内容怎样组织

文学史页面提供时间与区域坐标，作家和作品页面承担核心导读，阅读路径负责渐进次序，专题负责跨时代比较，理论与技巧则把抽象概念转化为可执行的细读问题。作品页的“阅读抓手”是一种建议方法，不是唯一解释。

## 来源与校订

事实信息优先核对档案馆、大学、图书馆、出版社、原始文献与专业研究资料。深度页面在页末公开列出主要来源和最近校订日期，但这表示完成了本站编辑流程，不等于经过匿名同行评审。遇到年代、版本或归属争议时，正文应说明不确定性，而不是制造精确结论。

## 工具使用

站点使用程序完成结构校验、关系检查、搜索索引和链接审计，也会使用语言模型协助资料整理与初稿生成。自动化工具不被视为事实来源；公开内容仍需依据页面列出的资料核对、重新组织并接受后续纠错。

## 版权边界

公开正文以概括、比较和阅读引导为目的，不提供受版权限制的作品全文，也不连续复制参考书、电子书或网页段落。私人研究资料、完整提取文本和页码映射不会进入公开构建产物。

## 更新与纠错

站点保留稳定 URL，并在结构、来源或事实发生重要变化时重新校订相关页面。若发现作品信息、译名、年代、链接或解释存在问题，请提交 [内容纠错](https://github.com/maaoding/literature-knowledge-base/issues/new?template=content-correction.yml)，注明页面地址、问题位置和可核对的依据。纠错入口用于具体内容问题，不收集私人阅读数据。
