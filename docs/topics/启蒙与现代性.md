# 启蒙与现代性

启蒙文学相信理性能够检验传统、改善社会并扩大个人自主，但文学很快发现，理性也可能变成计算、控制和自我欺骗。把《格列佛游记》《鲁滨逊漂流记》《浮士德》《呐喊》和《局外人》放在一起，可以看到现代个人从自信地改造世界，逐渐走向对自身位置和语言能力的怀疑。

## 专题坐标

十八世纪旅行叙事和冒险小说伴随商业、殖民与公共舆论扩张。个人被想象为能够观察、分类、劳动并积累财富的主体，同时也把陌生世界纳入自己的尺度。此后的现代文学继承了个人解放的目标，却更敏锐地看到城市、职业、民族国家和日常规范制造的新约束。

中国现代文学还面对语言变革。白话文、新小说和杂文不仅传播新思想，也重新规定谁可以说话、怎样描述普通人的生活，以及文学应当如何介入公共问题。

## 核心问题

- 理性是在破除偏见，还是可能产生新的傲慢和分类秩序。
- 劳动、自我教育与个人奋斗能否真正摆脱社会关系。
- 当传统价值失效后，现代人依据什么解释孤独、责任和荒诞。

## 比较路径

先读《格列佛游记》和《鲁滨逊漂流记》。前者用尺度变化和陌生社会讽刺欧洲理性的自满，后者通过孤岛劳动建立自主个人的神话。两者共享旅行框架，却分别显示观察世界和占有世界之间的张力。

《浮士德》把求知欲扩展成永不停息的行动冲动。现代主体不再满足于既有边界，但每次突破也会把代价转移给他人。由此进入《呐喊》，可以看到启蒙者在唤醒他人时遭遇语言失效、旁观和自我怀疑。

《局外人》进一步取消宏大的进步信念。人物与社会规则之间的断裂不再能够由奋斗解决，现代性问题转为：当意义不再天然存在，人是否仍能诚实地承担自己的处境。

## 阅读方法

阅读时区分作品中的“理性话语”和作品对它的反讽。记录人物如何命名他者、计算资源和解释成功，再观察被排除的人与经验。这样能避免把启蒙简单理解为从愚昧到进步的直线历史。

## 推荐入口

先从旅行和冒险叙事理解现代个人，再进入《浮士德》《呐喊》和《局外人》中的主体危机。
