# 启蒙与现代性

启蒙与现代性专题关注文学如何处理新语言、新个体、新社会结构带来的冲突。文学在这里不是单纯讲故事，而是在追问：人如何成为现代意义上的个体，又会为此付出什么代价。

## 关键问题

- 白话文和新文体怎样改变表达方式。
- 个人觉醒为什么常常伴随孤独、误解和失败。
- 文学如何在批判社会的同时反省自身语言。
