# 现代主义

现代主义关注现代经验中的陌生、断裂、异化和形式危机。它经常打破传统叙事的稳定感，让读者直接感受到人物解释世界的困难。

## 关键问题

- 人为什么会在家庭、城市、工作和制度中感到陌生。
- 叙事为什么不再总是线性、完整和透明。
- 文学形式本身如何表达现代经验。
