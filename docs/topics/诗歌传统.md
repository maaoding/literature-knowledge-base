# 诗歌传统

中国诗歌传统可以从三种声音进入：共同体歌唱、个人抒情、士人心境。它们分别对应《诗经》《楚辞》和唐宋诗词中的重要方向。

## 关键问题

- 谁在说话：共同体、个人、士人，还是历史见证者。
- 怎样抒情：比兴、象征、直抒、用典、景中寓情。
- 情感和秩序的关系：诗歌常在个人情绪和礼法、政治、历史之间摆动。
