# 女性与婚姻

女性与婚姻专题关注小说如何书写爱情背后的制度：财产、继承、家族、礼法、名誉和社会阶层。

## 关键问题

- 婚姻是感情选择，也是社会资源分配。
- 女性人物的判断力和行动空间常受制度限制。
- 家庭叙事可以揭示比公共历史更细密的权力关系。
