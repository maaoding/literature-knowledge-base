# 命运与伦理

命运与伦理专题关注人物在不可回避的选择、罪责、信仰和秩序中如何行动。它适合把戏剧、长诗和长篇小说放在一起读：不同文体都在追问人如何面对不可逃避的判断。

## 关键问题

- 人物是否真的拥有选择自由。
- 罪责来自个人行动，还是来自更大的制度和命运。
- 信仰、法律、家族和自我判断之间如何冲突。
