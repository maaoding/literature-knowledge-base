# 现实主义

现实主义不是把生活原样搬进作品，也不等于题材普通、细节很多。它通过人物关系、空间安排、金钱流动和叙述距离建立一个可分析的社会世界，让读者看到个人欲望如何被阶层、职业、家庭和制度塑造。《包法利夫人》《战争与和平》《安娜卡列尼娜》《罪与罚》和《呐喊》展示了这种方法在不同社会中的变化。

## 专题坐标

十九世纪城市化、市场扩张和公共生活变化，使小说成为观察社会关系的重要形式。人物不仅拥有性格，也拥有收入、债务、职业、住址和声誉；这些条件决定他们能进入哪些空间、认识哪些人，以及承担何种风险。

现实主义传入不同文学传统后并非简单复制欧洲模式。中国现代小说借短篇、白话和讽刺处理启蒙困境与乡土社会，二十世纪作家又吸收心理分析、象征和形式实验，使现实主义能够表现记忆、创伤和难以直接说明的经验。

## 核心问题

- 日常细节怎样显示人物背后的阶层、财产和权力关系。
- 叙述者是在公开评价人物，还是通过距离、反讽和视角让人物自行暴露局限。
- 社会环境能够解释行为到什么程度，又在何处不能替人物免除责任。

## 比较路径

先比较《包法利夫人》和《安娜卡列尼娜》。两部小说都写欲望与婚姻秩序的冲突，但叙述语气不同：福楼拜强调语言、幻想与消费如何相互强化，托尔斯泰则把人物放进家庭、社交和道德选择组成的广阔关系网。

《战争与和平》把现实主义尺度扩展到战争与历史。宏大事件并不只由少数英雄推动，命令、偶然、家庭生活和普通人的判断共同构成历史。转入《罪与罚》，社会空间收缩为街道、房间和心理辩论，贫困既是压力，也是人物危险理论的试验场。

《呐喊》用更短的篇幅处理旁观、麻木和启蒙失败。熟悉的日常语言可能保存偏见，叙述者的迟疑也成为现实的一部分。现实主义由此不只是观察他人，也反省观察者自身的位置。

## 阅读方法

阅读时建立一张人物资源表：收入、职业、住所、教育、家庭和社交网络。再标记关键场景由谁观察、哪些信息被省略。这样既能看见社会结构，也不会把作品简化成“环境决定一切”的社会学案例。

## 推荐入口

可先从《呐喊》和《包法利夫人》学习细节与反讽，再进入俄罗斯长篇小说的社会网络和伦理复杂性。
