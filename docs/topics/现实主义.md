# 现实主义

现实主义不是简单复制现实，而是通过人物、情节和叙事结构揭示社会关系、制度压力和个体处境。它的关键不在“像不像”，而在作品怎样组织细节，让看不见的社会结构变得可读。

## 关键问题

- 社会环境怎样限制人物选择。
- 日常细节怎样显示阶层、财产、职业和家庭关系。
- 叙述者是否在判断人物，还是让人物自己暴露困境。
