# 死魂灵

《死魂灵》让乞乞科夫购买已经死亡却仍登记在册的农奴，以此获取财产资格。看似聪明的骗局逐渐揭示整个社会都依靠名册、官阶和表面礼仪维持运转。

## 为什么读

- 理解怪诞与现实主义可以共同揭示制度真相。
- 观察人物性格如何通过住宅、饮食、谈话和交易方式显现。
- 进入俄国小说对官僚语言与精神空洞的长期批判。

## 阅读提示

注意叙述者频繁跑题、抒情和夸张。这种不稳定语气让小说在社会讽刺与国家想象之间保持张力。
