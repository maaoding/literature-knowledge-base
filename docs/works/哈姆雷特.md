# 哈姆雷特

《哈姆雷特》通过复仇故事呈现权力、犹疑、语言和人性之间的复杂冲突，是世界戏剧传统的核心作品之一。

## 为什么读

- 理解戏剧如何通过冲突推动人物。
- 观察独白、对白和舞台行动之间的关系。
- 进入文艺复兴文学中的人性问题。
