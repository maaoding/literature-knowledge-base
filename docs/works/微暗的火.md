# 微暗的火

《微暗的火》由一首长诗、编者前言、庞大注释和索引组成。注释者不断把诗歌据为己有，用自己的流亡王国故事覆盖作者生活，读者必须在互相矛盾的材料中重建关系。

## 为什么读

- 体验书籍装置本身如何成为小说结构。
- 理解不可靠叙述并不只是隐藏真相，也会主动制造世界。
- 观察流亡记忆、作者权力和阅读欲望如何彼此纠缠。

## 阅读提示

可以先按书中建议从诗读到注释，再反向核对索引和前言。不要急于确定唯一真相，先记录解释者如何暴露自己的需要。
