# 庄子

《庄子》是中国散文中极具想象力的一支。它用寓言、悖论、梦境和夸张形象，拆解固定判断，打开自由、逍遥和相对性的思想空间。

## 为什么读

- 理解古典散文并不只有论辩，也有高度文学化的想象。
- 进入“逍遥”“齐物”“梦蝶”等影响深远的思想意象。
- 学习寓言如何让抽象思想变得可感。
