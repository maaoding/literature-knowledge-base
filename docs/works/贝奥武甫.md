# 贝奥武甫

**作者：** 盎格鲁-撒克逊诗人

英雄先后对抗格伦德尔、其母亲和巨龙，晚年战斗让荣耀与死亡直接相遇。

## 为什么读

理解日耳曼英雄伦理与基督教书写如何在同一史诗中并存。

## 阅读重点

作品既赞美勇力，也不断用墓葬、哀歌和王国脆弱性限制英雄神话。

## 进入方式

比较青年胜利与老年最后一战，观察英雄价值如何随时间改变。
