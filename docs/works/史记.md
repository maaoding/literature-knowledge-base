# 史记

《史记》既是史学著作，也是中国叙事文学的重要源头。它通过本纪、世家、列传等结构，把人物命运、政治选择和历史判断组织起来。

## 为什么读

- 理解中国文学中的人物传记传统。
- 学习如何在简洁叙事中呈现性格、冲突和命运。
- 看见历史判断不是抽象议论，而常通过人物行动表现出来。

## 阅读提示

可以先从人物传记读起，比如英雄、刺客、游侠、将相等类型，再理解全书体例。
