# 虚构集

**作者：** [博尔赫斯](/authors/博尔赫斯)

作品常以书评、传记或学术报告开场，却逐步制造不存在的书和世界。

## 为什么读

用极短小说进入无限、作者身份和文本现实的哲学迷宫。

## 阅读重点

知识形式本身成为虚构工具，可靠叙述与伪造无法简单分开。

## 进入方式

一篇一篇慢读，并核对叙述者如何用细节让不可能设定显得可信。
