# 堂吉诃德

《堂吉诃德》以骑士小说为对象展开讽刺，同时让小说开始反思阅读、想象和现实之间的关系。

## 为什么读

- 理解现代小说的源头之一。
- 观察讽刺如何处理理想与现实。
- 学习小说如何把“阅读行为”变成叙事主题。
