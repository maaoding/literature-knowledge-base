# 到灯塔去

**作者：** [伍尔夫](/authors/伍尔夫)

一次未能成行的灯塔旅行跨越多年，在家庭成员和访客意识中留下不同形状。

## 为什么读

观察家庭日常、战争缺席和时间流逝如何通过多重意识显现。

## 阅读重点

中间部分用房屋变化压缩漫长时间，重大死亡只以边缘方式出现。

## 进入方式

跟随视角转移而不是寻找全知结论，留意同一人物被他人如何误读。
