# 呐喊

《呐喊》是进入现代中国小说的重要入口。它用短篇形式呈现个体困境、社会病灶和启蒙意识。

## 为什么读

- 篇幅短，适合现代文学入门。
- 能集中看到鲁迅小说的叙事角度和批判力度。
- 可与现实主义、启蒙和白话文运动一起理解。
