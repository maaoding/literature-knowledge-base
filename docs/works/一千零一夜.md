# 一千零一夜

《一千零一夜》以框架叙事组织众多故事。讲述本身在这里具有生存意义：故事延缓死亡，也保存经验、欲望、智慧和想象。

## 为什么读

- 理解框架叙事和连环故事的力量。
- 扩展对世界民间故事传统的认识。
- 观察奇遇、财富、计谋和欲望如何推动故事。
