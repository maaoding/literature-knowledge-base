# 奥德赛

《奥德赛》以归乡为核心结构，把冒险、诱惑、身份确认和家庭秩序组织在一起。

## 为什么读

- 理解“归乡叙事”的古典模型。
- 观察英雄不只依靠力量，也依靠智慧、忍耐和叙述能力。
- 与《伊利亚特》的战争史诗形成互补。
