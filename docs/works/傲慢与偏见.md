# 傲慢与偏见

《傲慢与偏见》表面是爱情与婚姻故事，深层则是关于判断力、阶层秩序、财产制度和女性选择的小说。

## 为什么读

- 篇幅适中，适合作为世界文学入门小说。
- 可以学习讽刺叙述如何在日常对话中发挥作用。
- 能清楚看到婚姻选择和社会结构之间的关系。

## 阅读提示

读这本书时，不要只追踪感情线。要注意人物如何评价他人，又如何修正自己的判断。
