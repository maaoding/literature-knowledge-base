# 百年孤独

《百年孤独》通过布恩迪亚家族的循环命运，把拉丁美洲历史、神话感和现代小说结构结合起来。

## 为什么读

- 理解魔幻现实主义的叙事方式。
- 观察家族、历史和记忆如何互相缠绕。
- 扩展现代小说阅读的世界文学视野。
