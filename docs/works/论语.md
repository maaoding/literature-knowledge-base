# 论语

《论语》不是通常意义上的小说或诗歌，但它是理解中国文学语言、伦理判断和士人传统的基础文本。它的语录体短小、含蓄，常以对话和片段留下解释空间。

## 为什么读

- 理解“言简意深”的古典表达方式。
- 进入仁、礼、君子、学与行等核心伦理词汇。
- 为后续阅读士人诗文、古文和小说中的伦理判断打底。
