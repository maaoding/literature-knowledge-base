# 战争与和平

《战争与和平》是宏大现实主义小说的代表。它把战争、家庭、爱情、精神成长和历史哲学放在一个巨大的叙事结构中。

## 为什么读

- 理解长篇小说如何同时处理个人和历史。
- 看见战争不是单纯背景，而是改变家庭与个体命运的力量。
- 学习现实主义如何通过日常细节建立宏大结构。

## 阅读提示

可以按人物线索分阶段读，不必一开始就试图记住所有人物。先抓皮埃尔、安德烈、娜塔莎等核心人物的精神变化。
