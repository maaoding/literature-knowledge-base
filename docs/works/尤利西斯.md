# 尤利西斯

《尤利西斯》把一天、城市、日常行动、神话结构和意识流压缩在一起，是二十世纪现代主义形式实验的高峰之一。

## 为什么读

- 理解现代主义为什么重视形式本身。
- 观察日常生活如何被神话结构重新组织。
- 进入意识流、拼贴和语言实验。
