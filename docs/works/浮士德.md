# 浮士德

《浮士德》通过灵魂契约、知识欲和无限追求，呈现现代人不断越界、不断渴望经验的精神结构。

## 为什么读

- 理解现代性的精神动力和危险。
- 观察知识、欲望、行动和救赎如何纠缠。
- 进入启蒙与浪漫主义之间的张力。
