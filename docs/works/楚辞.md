# 楚辞

《楚辞》是中国诗歌传统中极具个体色彩和浪漫想象的一支。它与《诗经》共同构成早期诗歌的两种重要面貌：一个偏向共同体歌唱和礼乐秩序，一个偏向强烈个人抒情和象征世界。

## 为什么读

- 理解中国文学中“抒情主体”的加强。
- 进入香草美人、远游、神灵等象征系统。
- 看见政治理想和个人命运如何转化为诗歌。

## 阅读提示

不要只把《楚辞》当作难懂古文。可以先抓意象和情感方向，再回头看典故与训诂。
