# 名著索引

名著索引收纳文学史上常见的经典入口，分为中国经典和世界经典两条书架。你可以用搜索、国别、难度和标签筛选作品，也可以从侧边栏按结构进入。

## 选择作品的顺序

- 先选难度：入门作品负责建立信心，进阶作品负责打开结构，挑战作品适合专题深读。
- 再选传统：史诗与神话、中国古典小说、十九世纪现实主义、二十世纪现代主义各自有不同读法。
- 再选主题：现实主义、现代主义、诗歌传统、女性与婚姻等主题可以串起不同国别。
- 最后回到文学史：读完单篇作品后，再看它所在的时代和文体位置。
