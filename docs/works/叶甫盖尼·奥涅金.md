# 叶甫盖尼·奥涅金

《叶甫盖尼·奥涅金》把爱情故事、社交生活、旅行见闻和写作评论编织进诗体小说。奥涅金的厌倦与塔季扬娜的成长，使浪漫想象不断接受现实时间的检验。

## 为什么读

- 理解俄国文学中“多余人”形象的早期结构。
- 观察叙述者如何与人物和读者保持活跃对话。
- 进入浪漫主义形式向现实主义社会观察转变的过程。

## 阅读提示

注意两封书信和两次拒绝之间的时间差。人物没有简单交换位置，他们对自由、责任和社会身份的理解都已经改变。
