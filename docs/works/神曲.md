# 神曲

《神曲》以地狱、炼狱和天堂的旅程组织宗教想象、伦理秩序、政治判断和个人命运。

## 为什么读

- 理解欧洲文学中的宏大伦理结构。
- 学习长诗如何组织象征、人物和知识系统。
- 进入中世纪文学与文艺复兴之间的转折地带。
