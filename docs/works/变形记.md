# 变形记

《变形记》用一个极端设定进入现代人的异化处境：主人公醒来后变成甲虫，但文本最令人不安的地方不是变形本身，而是家庭、工作和制度如何迅速重组对他的态度。

## 为什么读

- 篇幅短，适合作为现代主义入门。
- 可以直接感受异化、荒诞和冷静叙述之间的关系。
- 适合与现实主义小说比较：它不解释社会全貌，而呈现无法解释的处境。

## 阅读提示

不要急着寻找单一寓意。更重要的是观察：变形之后，谁在说话，谁在沉默，谁重新分配了家庭中的权力。
