# 老人与海

**作者：** [海明威](/authors/海明威)

老渔夫独自出海捕获巨鱼，却在返程中不断失去成果。

## 为什么读

用极简叙事理解行动、失败与不依赖胜负的尊严。

## 阅读重点

故事区分外部失败与内部败北，动作细节承担大部分伦理含义。

## 进入方式

少找象征对照表，多关注身体劳动、海洋观察和自我对话。
