# 少年维特之烦恼

**作者：** [歌德](/authors/歌德)

维特的书信记录爱情、自然感受和社会受挫，读者被直接拉入他的主观世界。

## 为什么读

理解强烈主观情感如何反抗社会规范，又怎样封闭为自我毁灭。

## 阅读重点

抒情魅力与视野局限并存，作品同时表现情感解放和自我封闭。

## 进入方式

区分维特的语言感染力与他对他人处境的理解能力。
