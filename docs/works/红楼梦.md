# 红楼梦

《红楼梦》通过贾府兴衰、人物群像和细密日常，呈现情感、家族制度与命运感之间的复杂关系。

## 为什么读

- 理解中国古典小说的高度成熟形态。
- 学习人物群像和日常细节的叙事力量。
- 观察诗词、礼法、情感和制度如何交织。
