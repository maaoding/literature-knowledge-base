# 失乐园

**作者：** [弥尔顿](/authors/弥尔顿)

史诗从撒旦战败写到亚当夏娃被逐，以宏大宇宙解释人类失乐。

## 为什么读

通过创世和堕落重写理解自由、服从与修辞魅力的危险关系。

## 阅读重点

撒旦的雄辩与行动后果形成强烈反差，读者必须主动判断语言的诱惑。

## 进入方式

先掌握事件线，再比较撒旦、亚当和夏娃各自如何解释自由。
