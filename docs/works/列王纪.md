# 列王纪

**作者：** [菲尔多西](/authors/菲尔多西)

史诗汇集创世、列王和英雄故事，以鲁斯塔姆等人物贯穿民族历史。

## 为什么读

从神话王朝到历史时代，理解波斯文化如何保存自身长时段记忆。

## 阅读重点

力量无法解除命运和亲族悲剧，英雄史同时也是权力循环的反思。

## 进入方式

可先选读鲁斯塔姆与苏赫拉布，再回到整体王朝结构。
