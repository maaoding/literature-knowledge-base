# 魔山

《魔山》把一座高山疗养院变成战前欧洲的观念实验室。主人公短暂停留的计划不断延长，重复生活逐渐改变时间尺度，各种思想也在封闭空间中争夺解释权。

## 为什么读

- 观察小说如何把时间感本身变成叙事对象。
- 理解疾病、教育和欧洲政治之间的隐喻关系。
- 比较现实主义细节与现代主义观念结构如何共存。

## 阅读提示

不必急于判断哪位辩论者代表作者。更值得追踪的是主人公如何在相互冲突的观念中形成经验，以及战争怎样突然终止漫长的山上时间。
