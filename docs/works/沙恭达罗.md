# 沙恭达罗

**作者：** [迦梨陀娑](/authors/迦梨陀娑)

爱情因诅咒和信物丢失而中断，最终通过记忆恢复走向重认。

## 为什么读

进入梵语诗剧的情味理论、自然描写和情感节奏。

## 阅读重点

自然环境和人物情绪相互渗透，戏剧重心是情感如何经历分离而成熟。

## 进入方式

关注关键场景之间的情绪递进，而不只把它当作误会解除的爱情故事。
