# 看不见的城市

《看不见的城市》以马可·波罗和忽必烈的对话为框架，把一座座城市组织为记忆、欲望、符号、死亡和连续性等主题序列。每篇短章可以独立阅读，又在严格排列中彼此回应。

## 为什么读

- 体验组合结构如何替代传统线性情节。
- 理解城市既是空间，也是语言、记忆和关系模型。
- 进入形式自觉明显但阅读门槛相对友好的后现代小说。

## 阅读提示

可以每次读一个主题组，并记录城市之间重复的物件和关系。全书真正描绘的不是城市名录，而是人怎样借叙述理解自身生活空间。
