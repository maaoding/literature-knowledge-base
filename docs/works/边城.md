# 边城

《边城》是中国现代文学中具有抒情气质的中篇小说。它写边地世界、人情、爱情和命运，也提供了不同于都市现代性的文学入口。

## 为什么读

- 理解现代文学中“乡土”和“抒情”两条线。
- 观察简洁叙述如何营造温柔而含蓄的情感。
- 与《呐喊》的批判性现代经验形成对照。
