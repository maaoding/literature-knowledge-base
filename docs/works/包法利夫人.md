# 包法利夫人

《包法利夫人》写浪漫幻想、婚姻生活、消费欲望和现实日常之间的冲突。它是理解现实主义叙述精确性的关键作品。

## 为什么读

- 观察欲望如何被阅读、消费和社会想象塑造。
- 理解冷静叙述如何产生强烈批判。
- 与《傲慢与偏见》《安娜·卡列尼娜》比较女性处境。
