# 源氏物语

《源氏物语》是日本古典文学的重要长篇物语。它通过宫廷生活、情感关系和细腻心理，展现物哀审美和贵族世界的情感秩序。

## 为什么读

- 扩展世界文学视野，不只以欧洲小说为中心。
- 理解宫廷叙事和心理描写的早期成熟形态。
- 与《红楼梦》比较家族、才情和情感世界。
