# 聊斋志异

《聊斋志异》用狐鬼花妖、书生、官吏和民间奇遇构成文言短篇世界。它的志怪外壳下，常有对欲望、婚姻、科举和社会不公的观察。

## 为什么读

- 进入中国文言短篇和志怪传统。
- 观察奇异故事如何承载现实批评。
- 理解短篇叙事中转折、讽刺和情感的力量。
