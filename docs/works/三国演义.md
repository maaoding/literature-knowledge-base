# 三国演义

《三国演义》把历史材料、民间讲述和章回小说结构结合起来，形成战争、谋略和人物类型高度鲜明的历史演义。

## 为什么读

- 理解中国古典小说中的历史演义传统。
- 观察忠义、谋略、天命和成败如何组织人物。
- 学习章回体如何通过回目、悬念和战役推进叙事。
