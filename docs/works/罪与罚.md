# 罪与罚

《罪与罚》围绕犯罪、惩罚、忏悔和信仰展开，把人物的心理冲突推到高压状态中。

## 为什么读

- 理解心理小说如何制造精神强度。
- 观察伦理问题如何进入叙事结构。
- 进入十九世纪俄国现实主义文学。
