# 坎特伯雷故事集

**作者：** [乔叟](/authors/乔叟)

朝圣队伍中的骑士、商人、修士和妇人以比赛方式讲故事。

## 为什么读

通过一群朝圣者的轮流讲述进入多阶层、多文类的中古社会。

## 阅读重点

每个故事都与讲述者身份发生关系，框架让文类和价值观彼此争辩。

## 进入方式

先读总序，再选三位社会位置不同的讲述者进行比较。
