# 变形记（奥维德）

**作者：** 奥维德

长诗从创世写到罗马时代，以身体和形态变化连接大量独立故事。

## 为什么读

以“变化”串联希腊罗马神话，理解后世艺术最重要的题材库之一。

## 阅读重点

变形可能是惩罚、逃脱、纪念或欲望后果，因此没有单一含义。

## 进入方式

按母题选读，再追踪同一故事如何被绘画、戏剧和现代小说改写。
