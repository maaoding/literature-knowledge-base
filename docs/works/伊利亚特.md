# 伊利亚特

《伊利亚特》以特洛伊战争为背景，但它真正聚焦的是英雄愤怒、荣誉、死亡和集体命运。

## 为什么读

- 进入西方史诗传统的核心源头。
- 理解英雄价值和战争秩序之间的紧张。
- 为后续阅读悲剧、史诗和现代重写打基础。
