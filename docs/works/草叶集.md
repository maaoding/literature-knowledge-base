# 草叶集

《草叶集》不是一次完成的固定诗集，而是惠特曼终生修订的诗歌工程。长句、列举和直接呼告把个体身体与城市、劳动、人群和大陆空间连接起来。

## 为什么读

- 理解自由诗如何用呼吸和语法组织节奏。
- 观察诗歌主体怎样从私人声音扩展为公共声音。
- 进入美国文学中的身体观、民主想象和现代城市经验。

## 阅读提示

先选短篇朗读，再进入长诗。遇到大量并列事物时，关注节奏如何改变，而不必急于为每个对象寻找象征解释。
