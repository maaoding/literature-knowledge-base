# 吉尔伽美什

**作者：** 佚名

乌鲁克王吉尔伽美什从暴烈英雄转变为死亡追问者，挚友之死改变了整部史诗的方向。

## 为什么读

从最早的英雄史诗进入死亡意识、友谊和文明记忆。

## 阅读重点

作品把武功、友谊、洪水和永生之草连接起来，最终承认人的有限性。

## 进入方式

可把前半的英雄行动与后半的精神旅行分开阅读，再观察两部分如何互相解释。
