# 西游记

《西游记》以取经结构连接神魔世界、修行过程和团队关系。它既通俗好读，又有丰富的宗教、寓言和成长意味。

## 为什么读

- 进入中国神魔小说和民间想象。
- 观察孙悟空等人物如何在冒险中被塑造。
- 理解一回一难的结构如何制造节奏。
