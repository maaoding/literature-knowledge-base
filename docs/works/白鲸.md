# 白鲸

《白鲸》以捕鲸船的追逐为主线，却不断插入鲸类知识、宗教联想、舞台独白和劳动细节。结构上的混杂让小说本身成为一次无法完成的解释行动。

## 为什么读

- 理解象征并非单一答案，而是相互冲突的解释集合。
- 观察个人执念如何把共同体拖入毁灭。
- 体验小说如何容纳知识、戏剧、史诗和冒险叙事。

## 阅读提示

不必把说明性章节视为情节障碍。它们展示了人试图分类世界的努力，也反衬白鲸始终无法被知识彻底占有。
