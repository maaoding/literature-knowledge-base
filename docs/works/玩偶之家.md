# 玩偶之家

**作者：** [易卜生](/authors/易卜生)

娜拉隐瞒的借款被揭露，丈夫的反应使理想家庭形象迅速崩塌。

## 为什么读

从一间客厅理解婚姻、名誉、法律和主体自由的制度冲突。

## 阅读重点

秘密不是偶然情节机关，而是暴露法律与家庭角色如何共同限制女性。

## 进入方式

注意海尔茂称呼娜拉的方式，以及结尾前后语言权力的变化。
