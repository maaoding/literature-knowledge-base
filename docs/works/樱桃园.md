# 樱桃园

《樱桃园》围绕庄园出售展开，却把真正的冲突藏在闲谈、回忆和不断错过的行动中。人物对同一片土地拥有不同时间感，因此也生活在彼此难以沟通的现实里。

## 为什么读

- 理解戏剧如何依靠潜台词而非正面冲突推进。
- 观察阶层变化怎样进入家庭记忆和日常动作。
- 体会喜剧节奏与失落感为何可以同时存在。

## 阅读提示

注意舞台声音、人物进出和话题中断。结尾的伐木声不仅标志财产转移，也让历史变化变得可听见。
