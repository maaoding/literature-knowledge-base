# 鲁滨逊漂流记

**作者：** 笛福

海难幸存者通过记录、劳动和管理资源建立孤岛秩序。

## 为什么读

从孤岛生活进入近代个人、财产、劳动和殖民意识的形成。

## 阅读重点

自立神话与殖民占有同时存在，个人成功依赖被自然化的权力关系。

## 进入方式

一面读生存细节，一面追问谁有权命名土地、资源和他人。
