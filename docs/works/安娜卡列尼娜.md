# 安娜·卡列尼娜

《安娜·卡列尼娜》把爱情、婚姻、家庭、社会判断和精神追求放在同一个现实主义结构中。

## 为什么读

- 理解个人欲望如何与社会规范发生冲突。
- 观察家庭伦理和精神寻找如何并行展开。
- 与《战争与和平》一起理解托尔斯泰的现实主义。
