# 水浒传

《水浒传》以梁山群像构成英雄传奇。它的重要性不只在人物数量，而在它塑造了江湖伦理、结义关系和反抗想象。

## 为什么读

- 观察群像小说如何逐步聚合人物。
- 理解民间英雄、官府秩序和江湖伦理之间的冲突。
- 学习章回小说中的人物出场和类型塑造。
