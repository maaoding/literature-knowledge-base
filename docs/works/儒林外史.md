# 儒林外史

《儒林外史》是中国古典讽刺小说的重要作品。它通过士人群像和科举世界，呈现名利、虚伪、人格变形和制度压力。

## 为什么读

- 理解讽刺小说如何用克制叙述呈现荒谬。
- 观察科举制度如何塑造人物价值。
- 与《红楼梦》的世情叙事形成对照。
