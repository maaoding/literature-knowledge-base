# 喧哗与骚动

**作者：** [福克纳](/authors/福克纳)

康普生家族的衰败由不同叙述者和不同日期重新讲述。

## 为什么读

通过四种视角感受家族历史如何在记忆、偏见和语言中破碎。

## 阅读重点

事件本身不变，意义却随意识结构改变，时间成为人物无法摆脱的内部压力。

## 进入方式

首次阅读可借人物表定位事件，不必强迫自己立即理清所有跳跃。
