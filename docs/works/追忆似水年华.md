# 追忆似水年华

《追忆似水年华》把记忆、时间、感官经验和社会世界组织成庞大的现代小说结构。它的核心不是外部情节，而是意识如何重新发现过去。

## 为什么读

- 理解现代小说如何把记忆变成叙事材料。
- 观察时间不只是背景，而是主题本身。
- 学习慢速、回旋式阅读。
