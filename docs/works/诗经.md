# 诗经

《诗经》是中国诗歌传统的重要源头，包含风、雅、颂等部分。它既能作为文学作品阅读，也能帮助理解礼乐、政治和日常生活之间的关系。

## 为什么读

- 建立中国文学史的起点意识。
- 理解抒情、比兴和民歌传统。
- 为后续阅读唐诗、宋词和古典小说打基础。
