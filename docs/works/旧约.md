# 旧约

**作者：** 古希伯来文献汇编

它汇集神话、律法、历史、预言、诗歌和智慧文学，内部文类极为丰富。

## 为什么读

理解西方文学中创世、出走、受难和救赎等基本叙事代码。

## 阅读重点

共同体通过契约解释历史，也在诗篇、哀歌和约伯的抗辩中保存复杂情感。

## 进入方式

建议从《创世记》《出埃及记》《诗篇》《约伯记》和《雅歌》分组进入。
