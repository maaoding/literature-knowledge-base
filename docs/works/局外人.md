# 局外人

《局外人》用冷静、短促的叙述呈现荒诞、审判和现代个体的疏离感。它的难点不在情节，而在人物与社会意义系统之间的断裂。

## 为什么读

- 篇幅短，适合进入二十世纪文学。
- 理解荒诞感如何通过叙述语气呈现。
- 与《变形记》比较现代人的异化经验。
