# 雪国

**作者：** [川端康成](/authors/川端康成)

东京男子反复前往雪国，与艺伎驹子建立亲近又不对等的关系。

## 为什么读

在传统审美、感官片段和现代距离感之间阅读一段无法抵达的关系。

## 阅读重点

镜像、雪景和声音比明确心理解释更重要，人物始终隔着审美化视线。

## 进入方式

留意谁在观看、谁被观看，以及美感如何同时遮蔽现实生活。
