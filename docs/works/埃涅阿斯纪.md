# 埃涅阿斯纪

**作者：** [维吉尔](/authors/维吉尔)

特洛伊幸存者埃涅阿斯历经漂泊与战争，为未来罗马寻找土地。

## 为什么读

看荷马史诗如何被改写为罗马建国、使命与牺牲的叙事。

## 阅读重点

公共使命要求他放弃私人幸福，史诗不断记录这种选择造成的损失。

## 进入方式

把迦太基爱情段落与后半战争对读，观察使命语言如何改变人物。
