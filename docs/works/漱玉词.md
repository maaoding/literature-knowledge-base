# 漱玉词

《漱玉词》所代表的李清照词作，以精确节奏和日常意象承载复杂时间感。酒、花、雁、帘幕和梧桐并非固定符号，而会随生活处境改变情绪重量。

## 为什么读

- 体会词的声音和停顿如何直接参与意义。
- 比较早期生活感受与南渡之后的离乱经验。
- 进入中国文学中高度成熟的女性自我书写。

选取不同时期的作品对读，先朗读并标记节奏，再观察相似意象如何从闲愁转向历史和生命的失落。
