# 米德尔马契

《米德尔马契》同时追踪多组人物，把婚姻选择、医学改革、政治变化、财产关系和个人理想编织在一座小城中。每个人的决定都会通过关系网络改变他人的可能性。

## 为什么读

- 观察现实主义如何表现复杂社会，而不依赖单一英雄。
- 理解同情、判断和责任如何共同构成伦理叙事。
- 比较女性理想、婚姻制度与知识追求之间的限制。

## 阅读提示

可以为主要人物记录“愿望、限制、影响”三项。随着支线汇合，会逐渐看见小说如何把偶然事件转化为社会因果。
