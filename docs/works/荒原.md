# 荒原

《荒原》把都市对话、神话、宗教文本和多种语言并置，让阅读过程不断遭遇声音与场景的断裂。碎片并非等待恢复的完整故事，而是作品面对现代经验的基本形式。

## 为什么读

- 理解拼贴和多声部如何组织现代诗歌。
- 观察古老神话怎样被用来测量当代生活的贫乏。
- 体会节奏、重复和水意象如何在碎片之间建立联系。

## 阅读提示

第一遍先跟随声音和意象，不必停下来查完所有典故；第二遍再利用注释辨认引文来源，比较知识增加后结构感如何变化。
