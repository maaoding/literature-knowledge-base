# 十九世纪世界文学

这是十九世纪文学的跨流派导读页。浪漫主义并未在现实主义兴起后突然消失；象征主义和唯美主义也在现实主义仍然强盛时出现。三者共同回应工业社会、民族国家和现代城市。

## 历史位置

- [浪漫主义时期](/history/浪漫主义文学)：想象、自然与政治自由。
- [现实主义与自然主义](/history/现实主义与自然主义)：阶层、家庭、城市与心理观察。
- [世纪末的西方文学](/history/世纪末西方文学)：象征、唯美和现代主义前夜。
- [世纪之交的东方文学](/history/世纪之交东方文学)：日本和印度的文学现代化。

## 阅读重点

- 现实主义：文学如何组织社会细节，而不是简单复制现实。
- 家庭伦理：婚姻、继承、财产和情感关系如何塑造人物选择。
- 心理深度：罪感、信仰、欲望和理性如何在人物内部冲突。
- 形式转向：外部社会观察怎样逐步让位于象征、感觉和潜意识。
