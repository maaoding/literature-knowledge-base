# 十九世纪世界文学

十九世纪世界文学的重要面貌，是小说成为理解现代社会的强大工具。现实主义小说通过家庭、财产、阶层、城市和战争，把个人命运放入更复杂的社会结构中。

## 历史位置

欧洲社会结构、资本关系和现代国家经验快速变化，小说也从冒险传奇、讽刺叙事转向更细密的社会观察。人物不再只是道德类型，而是在家庭、制度、阶层和心理压力中行动。

## 阅读重点

- 现实主义：文学如何组织社会细节，而不是简单复制现实。
- 家庭伦理：婚姻、继承、财产和情感关系如何塑造人物选择。
- 心理深度：罪感、信仰、欲望和理性如何在人物内部冲突。
