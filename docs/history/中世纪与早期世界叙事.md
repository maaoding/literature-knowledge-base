# 中世纪与早期世界叙事

这是一个跨区域导读页。详细文学史已经拆分为欧洲、阿拉伯、波斯、日本及亚洲四条线，便于比较信仰、宫廷、城市与口头故事如何生成不同文类。

## 区域入口

- [中古欧洲文学](/history/中古欧洲文学)：英雄史诗、骑士文学、宗教长诗和城市故事。
- [中古阿拉伯文学](/history/中古阿拉伯文学)：诗歌竞争与开放的框架叙事。
- [中古波斯文学](/history/中古波斯文学)：民族史诗、四行诗和苏菲传统。
- [中古日本和其他亚洲国家文学](/history/中古日本和亚洲文学)：和歌、物语、能剧与俳句。

## 阅读重点

横向阅读时，可以比较《神曲》的伦理宇宙、《一千零一夜》的连续讲述、《列王纪》的民族记忆和《源氏物语》的时间感。它们共同说明中古文学远比单一的欧洲“黑暗时代”更加多元。
