# 现代主义文学的兴起

**时代坐标：约 1890 年至 1945 年。** 都市速度、世界大战、新物理学和心理学动摇了稳定现实观。现代主义作家不再假定世界能够被透明叙述，而是把形式本身变成问题。

## 后期象征主义：神话的复活与重构

象征主义拒绝直接说明，强调意象、声音和暗示。叶芝、艾略特等诗人重新调用古代神话，并非复古，而是借共同结构整理现代碎片。

## 未来主义：速度与动力之美

未来主义赞美机器、城市和冲击，宣言式写作主动破坏传统语法和审美礼仪。它展示现代艺术与技术激情的结合，也暴露艺术被暴力政治吸引的风险。

## 表现主义：“挤压”与变形的艺术

表现主义不追求外部世界的准确比例，而让空间、身体和语言按照内心压力变形。[卡夫卡](/authors/卡夫卡)的寓言式小说使制度恐惧成为冷静日常。

## 意识流小说：时间与叙事

[普鲁斯特](/authors/普鲁斯特)、[乔伊斯](/authors/乔伊斯)和[伍尔夫](/authors/伍尔夫)分别从记忆、城市语言和日常感知改造长篇小说。[《尤利西斯》](/works/尤利西斯)把一天扩展成语言宇宙，[《到灯塔去》](/works/到灯塔去)则让时间在家庭意识间流动。

## 达达主义与超现实主义：心理自动性

达达用偶然和反艺术回应战争理性的破产，超现实主义尝试释放梦、欲望和无意识。两者都把创作过程从控制严密的作品转向事件、游戏和自动书写。
