# 文学史

这个栏目用统一年代组织中国与世界文学的两条轨迹。中国线从先秦、魏晋、唐宋、明清延伸到现代文学；世界线从两河、埃及、希伯来、印度、希腊和罗马出发，经中古区域传统、近代欧洲转型，进入现代主义、拉美、亚非与移民文学。两条线并列而不互相套用分期标签。

## 阅读方法

文学史不要只背朝代和流派，可以抓三条问题线：

- 文体怎样变化：史诗、诗歌、散文、戏剧、长篇小说分别在什么时候成为主导形式。
- 人怎样被书写：从礼乐共同体、士人自我、家族伦理，到现代个体和异化经验。
- 社会怎样进入文学：战争、制度、城市、阶层、家庭和现代性如何改变叙事。
