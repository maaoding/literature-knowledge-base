# 文学史

这个栏目用时间线组织文学的发展脉络。首版按“中国文学内部演变”和“世界文学关键转向”两条线并置：先秦源头、魏晋文学自觉、唐宋诗词、明清小说、现代文学、十九世纪现实主义、二十世纪现代主义。

## 阅读方法

文学史不要只背朝代和流派，可以抓三条问题线：

- 文体怎样变化：诗歌、散文、戏剧、小说分别在什么时候成为主导形式。
- 人怎样被书写：从礼乐共同体、士人自我、家族伦理，到现代个体和异化经验。
- 社会怎样进入文学：战争、制度、城市、阶层、家庭和现代性如何改变叙事。
