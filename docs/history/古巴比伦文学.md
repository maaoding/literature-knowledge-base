# 古巴比伦文学

**时代坐标：约公元前 3500 年至前 539 年。** 两河流域的泥板文字让早期神话和英雄叙事得以保存。这里的文学始终面对不稳定的自然、城市秩序与人的有限生命。

## 神话与仪式

创世神话把宇宙理解为从混沌走向秩序的过程。神的战斗既解释天地和河流的形成，也与城邦节庆、王权和农业周期结合。神话在这里不是孤立的故事，而是共同体反复表演和确认世界秩序的方式。

## 《吉尔伽美什》：追寻永生

[《吉尔伽美什》](/works/吉尔伽美什)从英雄武功转向死亡恐惧。恩启都之死迫使主人公放弃单纯的征服，踏上寻找永生的旅程。最终的失败让史诗得出更成熟的认识：人不能取消死亡，却可以通过友谊、城市和叙事留下痕迹。

## 文学史位置

洪水、英雄历险、友谊和冥界旅行等母题在西亚与地中海传统中持续变形。古巴比伦文学因此不仅是“最早”的文献，更提供了后来史诗反复使用的叙事原型。
