# 古罗马文学

**时代坐标：约公元前 3 世纪至公元 5 世纪。** 罗马文学大量吸收希腊形式，却并非被动模仿。共和国政治、帝国扩张和拉丁语言使史诗、讽刺与爱情诗获得新的历史重量。

## 哲理诗人与爱情诗人

卢克莱修用长诗讨论自然和人的恐惧，贺拉斯把节制、友谊与诗艺结合，卡图卢斯和奥维德则发展出更鲜明的私人情感。罗马诗歌在公共责任和个人欲望之间形成持续张力。

## 文人史诗与神话故事诗

[维吉尔](/authors/维吉尔)的[《埃涅阿斯纪》](/works/埃涅阿斯纪)把特洛伊流亡者写成罗马起源的承担者，个人爱情必须让位于建国使命。奥维德的[《变形记》](/works/奥维德变形记)则以“变化”串联大量神话，让世界成为不断越界和改形的叙事网络。

## 文学史位置

罗马文学把希腊遗产传递给中世纪和文艺复兴，并确立拉丁语的教育权威。后世欧洲史诗、爱情诗、讽刺文学和神话改写都不断返回这一传统。
