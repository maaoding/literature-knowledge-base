# 启蒙与浪漫主义

这是连接两种现代精神的导读页。启蒙文学相信公共理性可以批判偏见和专断，浪漫主义则提醒人们：自然、情感、想象和历史记忆不能被工具理性完全吸收。

## 分期入口

- [启蒙运动时期](/history/启蒙运动文学)：百科全书、讽刺文学与近代小说兴起。
- [浪漫主义时期](/history/浪漫主义文学)：自然、反抗、民族意识与现代艺术家。
- [文艺复兴与宗教改革时期](/history/文艺复兴与宗教改革)：现代自我与读者市场的更早起点。

## 阅读重点

阅读[《浮士德》](/works/浮士德)时，两股力量会在同一人物身上相遇：他既继承启蒙的求知冲动，又以浪漫主义方式追逐无限。这个冲突一直延伸到现代文学。
