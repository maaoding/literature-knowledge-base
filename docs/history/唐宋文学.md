# 唐宋文学

唐宋文学是中国古典文学高度成熟的阶段。唐诗把个人情感、时代经验和语言形式推向高峰，宋词则在日常情绪、都市生活和精细化抒情中展开新的空间。

## 重点入口

- 唐诗：从李白、杜甫、王维等作家理解盛唐气象、诗史意识和山水审美。
- 宋词：从婉约、豪放和都市生活理解抒情方式的变化。
- 古文运动：理解文体、思想和士人精神之间的关系。
