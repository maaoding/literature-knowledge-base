# 二十世纪现代主义

这是 20 世纪形式实验的跨章节导读页。现代主义不是唯一主线，它与深化的现实主义、战后荒诞文学、拉美小说和亚非民族文学同时存在。

## 历史位置

- [现实主义文学的深化](/history/现实主义文学深化)：战争、反乌托邦与迷惘的一代。
- [现代主义文学的兴起](/history/现代主义文学兴起)：象征、表现、意识流与超现实主义。
- [从现代主义到后现代主义](/history/现代主义到后现代主义)：荒诞派、黑色幽默与元小说。
- [拉美文学的爆炸](/history/拉美文学爆炸)：神话时间、殖民历史和现代小说实验。

## 阅读重点

- 异化经验：人面对家庭、组织、制度和自我时的陌生感。
- 形式实验：叙事时间、视角和语言不再透明。
- 历史创伤：个人记忆和集体历史常以非现实方式返回。
- 多中心传播：形式如何在拉美、亚洲和非洲获得不同的历史含义。
