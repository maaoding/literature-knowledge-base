# 二十世纪现代主义

二十世纪文学面对城市化、战争、制度压力和个体精神危机，逐渐把注意力转向异化、意识、时间结构和叙事形式。现代主义不是一种单一风格，而是一组对现代经验的回应。

## 历史位置

现实主义关注社会整体结构，现代主义则更强调现代个体如何感到陌生、断裂和失控。叙事也不再总是线性展开，而会使用寓言、意识流、碎片化、循环时间等方式。

## 阅读重点

- 异化经验：人面对家庭、组织、制度和自我时的陌生感。
- 形式实验：叙事时间、视角和语言不再透明。
- 历史创伤：个人记忆和集体历史常以非现实方式返回。
