# 汉代文学

汉代文学承接先秦典籍与诸子散文，又在统一帝国的制度、地域和知识结构中形成新的表达规模。辞赋铺陈都城、物产与秩序，史传散文以人物行动组织历史判断，乐府诗则把战争、劳作、婚姻和离别带入可传唱的语言。

## 时代坐标

从西汉建立到东汉结束，政治统一与边疆扩展改变了文学观察世界的尺度。经学进入制度中心，史官传统得到重组，民间歌辞也通过采集、整理和传播进入书面文学。

## 主要文体

- 辞赋：以铺陈、问答和宏阔描写表现帝国空间，也逐渐转向个人遭际。
- 史传散文：通过本纪、世家和列传塑造人物，使历史判断落实到选择与命运。
- 乐府诗与古诗：把口语节奏、社会生活和个人抒情带入诗歌传统。

## 核心作家与作品

- [司马迁](/authors/司马迁)与[《史记》](/works/史记)：历史著述与叙事文学的关键交汇点。
- 汉乐府与《古诗十九首》：从公共经验走向更细密的生命意识和离别书写。

## 与前后时期的关系

汉代文学把先秦的思想、史学和歌谣资源转化为成熟文体，又为魏晋南北朝的文学自觉、五言诗发展和个体抒情准备了语言基础。
