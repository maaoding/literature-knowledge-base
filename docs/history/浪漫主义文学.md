# 浪漫主义时期

**时代坐标：约 1790 年至 1850 年。** 革命、工业化和民族国家兴起改变欧洲经验。浪漫主义用想象、自然、历史和强烈个体性回应机械理性，同时也创造现代艺术家形象。

## 德国：从浪漫主义到民族主义

德国浪漫派重视梦、夜、童话和碎片形式，并从民间故事中寻找民族语言。[歌德](/authors/歌德)的[《浮士德》](/works/浮士德)跨越多个时期，把无限求知和现代扩张的代价写入诗剧。

## 英国：自然的歌手与激进的流亡诗人

华兹华斯从普通生活和自然经验寻找诗意，柯勒律治发展超自然想象。[拜伦](/authors/拜伦)和雪莱把诗歌与政治反抗结合，“拜伦式英雄”成为骄傲、孤独和反叛的现代类型。

## 法国：浪漫主义与政治自由

[雨果](/authors/雨果)推动戏剧突破古典规则，并在[《巴黎圣母院》](/works/巴黎圣母院)中把历史建筑、边缘人物和社会暴力组织成宏大画面。法国浪漫主义与革命记忆和自由主义政治紧密相连。

## 其他欧洲国家：民族主义与民主主义

在东欧、意大利和北欧，搜集民歌、重写历史与争取民族独立相互促进。文学既发现地方传统，也参与想象新的政治共同体。

## 美国：新大陆的声音

爱默生和梭罗强调精神独立，霍桑与爱伦·坡探索罪、幽暗心理和形式控制，惠特曼则用开放诗行歌唱身体、民主与大陆空间。美国文学由此开始建立区别于欧洲的声音。
