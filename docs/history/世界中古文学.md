# 世界中古文学

中古文学不是古代与近代之间的空白。欧洲的修道院、宫廷和城市，阿拉伯世界的诗歌与故事市场，波斯的民族史诗与苏菲传统，日本的和歌、物语和能剧，都在这一时期形成稳定的文类和审美规范。

## 四个区域

- [中古欧洲文学](/history/中古欧洲文学)：英雄、骑士、圣徒和市民共同进入俗语叙事。
- [中古阿拉伯文学](/history/中古阿拉伯文学)：诗歌传统与《一千零一夜》的开放故事网络。
- [中古波斯文学](/history/中古波斯文学)：民族记忆、人生感叹与神秘主义诗歌。
- [中古日本和其他亚洲国家文学](/history/中古日本和亚洲文学)：和歌、物语、能剧与俳句的连续发展。

## 横向阅读

这一时期最值得关注的是不同文化如何处理“信仰与世俗生活”的关系。欧洲骑士爱情、波斯苏菲诗和日本物哀虽然背景不同，却都让欲望在严格秩序中寻找表达空间。城市成长又推动故事、讽刺和日常人物进入文学。
