# 世纪末的西方文学

**时代坐标：约 1850 年至 1910 年。** 进步信念与衰败感觉并存。都市生活、商品文化、帝国扩张和新心理学让作家怀疑现实主义能够完整解释现代经验。

## 法国：世纪末的情绪体验

波德莱尔的[《恶之花》](/works/恶之花)把都市震惊、厌倦和感官对应写入诗歌。魏尔伦、兰波与马拉美继续削弱直白陈述，让声音、暗示和语言自身成为诗的中心。

## 英国：唯美主义、命运观与帝国作家

唯美主义强调艺术不必服从道德功用，王尔德以悖论和戏剧揭露社会表演。哈代笔下的个人则受到环境、偶然和习俗挤压，帝国作家又把殖民世界带入英语叙事。

## 德国和奥地利：非理性哲学和心理学

尼采、精神分析和维也纳现代文化改变了人物观。梦、欲望、压抑和不可靠自我逐渐成为文学材料，为表现主义和意识流小说准备条件。

## 文学史位置

世纪末文学是现实主义与现代主义之间的转换地带。它没有放弃现代生活，而是改用象征、碎片和心理深描表现难以被外部观察捕捉的经验。
