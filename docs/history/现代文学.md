# 现代文学

现代文学的关键变化是白话文成为主要表达工具，小说、杂文、新诗等形式被用来回应个人觉醒、社会变革和历史压力。

## 重点入口

- 白话小说：以新的语言处理现代人的处境。
- 杂文：在公共议题、思想批评和文学表达之间建立锋利关系。
- 新诗：尝试用新的节奏和句法表达现代经验。
