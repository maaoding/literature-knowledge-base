# 先秦文学

先秦文学是中国文学的源头阶段，核心不只在作品数量，而在它建立了后世不断回到的表达范式：诗歌的抒情、诸子散文的辩论、历史叙事的伦理判断。

## 重点入口

- 《诗经》：理解诗歌、礼乐和民间歌谣传统的起点。
- 诸子散文：理解先秦思想如何通过寓言、辩难和论说进入文学表达。
- 历史叙事：理解人物、事件与伦理判断之间的关系。

## 推荐阅读

先读 [《诗经》](/works/诗经)，再回到诸子散文和历史叙事，会更容易把“文学源头”和“思想源头”连在一起。
