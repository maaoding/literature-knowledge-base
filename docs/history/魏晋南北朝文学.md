# 魏晋南北朝文学

魏晋南北朝是中国文学从经学、政治功能中逐渐分化出审美自觉的重要阶段。这个时期的文学不只是朝代夹缝中的过渡，而是形成了对个体风度、生命短促、自然审美和语言形式的持续关注。

## 历史位置

这一阶段上承先秦两汉的诗文传统，下启唐诗的成熟格局。士人面对政治动荡和生命无常，逐渐把个人气质、自然感受和精神选择写入文学。

## 阅读重点

- 文学自觉：文学开始被作为独立审美活动讨论。
- 生命意识：短暂、无常、归隐和自由成为高频主题。
- 志怪与想象：鬼神、异闻、奇遇扩大叙事空间。
