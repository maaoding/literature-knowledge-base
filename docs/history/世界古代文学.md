# 世界古代文学

古代文学不是一条只从希腊出发的直线。两河流域、埃及、希伯来、印度、希腊与罗马分别用神话、宗教诗歌、史诗和戏剧回答相近的问题：世界怎样开始，人为什么死亡，个人如何面对神意，群体又怎样保存共同记忆。

## 六个入口

- [古巴比伦文学](/history/古巴比伦文学)：从创世神话和《吉尔伽美什》进入死亡意识。
- [古埃及文学](/history/古埃及文学)：理解复活信仰、颂诗和亡灵书写。
- [古希伯来文学](/history/古希伯来文学)：观察民族历史怎样被组织为圣书和诗歌。
- [古印度文学](/history/古印度文学)：从颂神诗、两大史诗和梵语戏剧理解达摩与情味。
- [古希腊文学](/history/古希腊文学)：从英雄史诗转向城邦悲剧和喜剧。
- [古罗马文学](/history/古罗马文学)：看希腊形式如何被帝国经验和拉丁语言重新塑造。

## 比较线索

可以抓住三组问题横向阅读：巴比伦与埃及都反抗死亡，希伯来与印度都以神圣文本维系秩序，希腊与罗马则把神话加工为高度自觉的文学形式。它们之间存在传播和改写，却不能被压缩成单一源头。
