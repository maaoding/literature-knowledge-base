# 世界近代文学

从 14 世纪到 20 世纪初，文学逐渐从宗教共同体和宫廷秩序中分化出来。印刷、城市、民族国家和读者市场改变了作品的生产方式，“个人”也从被秩序安排的角色变成能够欲望、怀疑、反抗并观察社会的主体。

## 阅读方式

近代文学可以按“人的解放、理性的制度化、情感的反拨、社会的再发现”四步理解。各种主义并非简单接力，浪漫主义想象与现实主义观察经常在同一作家身上并存。
