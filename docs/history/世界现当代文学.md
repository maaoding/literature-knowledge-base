# 世界现当代文学

20 世纪文学面对两次世界大战、革命、殖民体系瓦解、城市化和跨国迁徙。现实本身变得难以用稳定因果解释，作家既深化社会观察，也不断实验时间、意识、语言和叙述者的位置。

## 六个方向

- [现实主义文学的深化](/history/现实主义文学深化)：战争、反乌托邦和迷惘的一代。
- [现代主义文学的兴起](/history/现代主义文学兴起)：象征、表现、意识流与超现实主义。
- [从现代主义到后现代主义](/history/现代主义到后现代主义)：存在主义、荒诞派、黑色幽默和元小说。
- [拉美文学的爆炸](/history/拉美文学爆炸)：殖民历史、神话时间和小说实验。
- [亚非文学的复兴](/history/亚非文学复兴)：民族独立、本土语言与现代形式。
- [流亡作家与移民文学](/history/流亡与移民文学)：跨语写作、身份重构与后殖民世界。

## 多中心文学史

现当代文学不再适合只用欧洲流派接力来理解。拉丁美洲、亚洲、非洲和移民作家改变了“中心”的含义。时间线因此既保留现代主义形式史，也展示反殖民、民族重建和迁徙经验。
