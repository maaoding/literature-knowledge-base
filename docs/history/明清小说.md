# 明清小说

明清时期，长篇章回小说和世情小说高度成熟。文学的中心不再只在诗文，也转向复杂叙事、人物群像、城市生活、家族结构和日常伦理。

## 历史位置

如果说唐宋文学把诗词推到高峰，明清小说则把“讲故事”发展成大型结构。章回体、人物群像、世情描写和讽刺传统，为后来的现代小说提供了重要资源。

## 阅读重点

- 叙事结构：章回、线索、伏笔和人物关系网。
- 日常世界：饮食、礼法、财产、婚姻、官场和家族。
- 世情判断：小说常在日常细节中呈现制度和人情。
