# 古埃及文学

**时代坐标：约公元前 3000 年至公元 1 世纪。** 尼罗河的周期、太阳运行和王权仪式共同塑造了古埃及人对生命循环的理解，文学也因此不断连接死亡与再生。

## 死亡与复活

奥西里斯神话把自然荣枯、身体破碎和灵魂复活组织成完整模式。保存身体、建造陵墓和为亡者准备语言，不只是葬俗，也说明古埃及文化相信文字能帮助个体穿越死亡。

## 《亡灵书》：反抗死亡

[《亡灵书》](/works/亡灵书)不是单一作者一次写成的作品，而是长期积累的经文、颂歌和咒语集合。它为亡灵提供通往审判之所的知识，同时反复确认名字、心脏和自我认同的重要性。其文学价值正在于把抽象信仰变成旅程、问答和形象。

## 文学史位置

与巴比伦史诗的英雄求生不同，埃及文学更强调仪式、语言和秩序保证的再生。二者共同显示，死亡意识是古代文学形成自我意识的重要动力。
