# 认知叙事学

认知叙事学关注故事与心智的双向关系：读者凭什么从零散文字建立一个可活动的世界，故事又怎样训练、扰乱或扩展我们理解时间、他人与自身经验的方式。它把阅读看成持续推断，而非被动接收完整信息。

## 问题起点

文本不会逐项说明房间每个角落、人物全部动机或事件之间所有空档，读者却常觉得自己“看见”了完整场景。我们利用生活知识、体裁习惯和已读信息补足空白，也可能过早套用类型，把人物和因果理解错。分析要问：某个推断由哪些文字触发，依赖何种图式，后续材料为何确认或迫使我们修正它？

## 核心概念

**图式**是组织常见情境和行动序列的知识结构，能降低理解负担，也会遮蔽不符合预期的经验。**故事世界**是读者依据话语线索建构的时空、人物与规则整体。理解人物需要进行心智归因，即推测其信念、欲望和知识边界；理解空间则会形成**认知地图**，把地标、路线和视角连接起来。认知风格还可指作品反复要求读者采用的注意与推理方式。

## 分析步骤

先标出文本明确提供的感知、行动和空间线索，再写下自己自动补入的内容。第二步识别被启动的日常或体裁图式，并寻找不匹配处。第三步追踪人物心智归因：谁知道什么，我们为何相信其意图，是否把读者知识误给人物。分析空间时画出路线和地标，记录地图在哪些段落失效。最后说明作品是帮助读者自然化陌生经验，还是故意保持认知困难。

## 作品试读

《局外人》以简短陈述削弱常见的情感解释线索，读者容易用社会脚本补足默尔索，又不断遭遇不匹配。《追忆似水年华》让感官触发记忆，过去不是存档重放，而在当下知觉中重组。《看不见的城市》提供彼此矛盾又可类比的城市描述，读者建立的地图更像概念网络，而不是可按比例测量的地理图。

## 方法边界

认知概念不能把文学效果化约为普遍脑机制，也不能把研究者的阅读反应冒充所有读者的心理事实。图式具有文化和历史差异，实验发现与单篇细读的证据层级也不同。这个方法擅长解释推断过程，却可能低估制度、权力和语言传统，因此应明确哪些判断来自文本，哪些只是关于读者处理方式的假设。

阅读同时包含自下而上的线索处理与自上而下的知识调用。一个陌生细节可能迫使读者修正已有图式，一个强势体裁预期也可能让读者忽略反证。认知分析不能只罗列心理概念，而要说明两种方向在具体段落怎样竞争：哪些词触发推断，哪些新信息让人物模型重新分类，读者又在何时意识到自己的常识并不适用。

若要提出“读者通常如何反应”的结论，还应区分文本分析与经验研究。单次细读只能证明作品提供了某些线索和处理可能；问卷、眼动或阅读实验才能支持群体性主张。《局外人》《追忆似水年华》和《看不见的城市》分别检验情感脚本、记忆模型与空间建构，认知困难并非缺少信息，而可能是作品精确控制线索的结果。

## 继续阅读

“叙事时间与空间”提供描述文本安排的形式术语，“读者反应与接受史”把个体加工扩展到历史读者共同体。练习时选择一处初读误判，按“原始线索、调用图式、形成推断、后文修正”四步复盘，检验误判究竟由作品设计还是个人经验造成。
