# 明清小说入门

<ReadingPathGoal />

## 路径说明

明清小说不能只用“四大名著”概括。这条路径把章回长篇与文言短篇并置，依次观察历史、江湖、神魔、志怪、讽刺和世情六种叙事能力。类型之间会互相借用，但各自组织人物和判断现实的方法不同。

阅读顺序从情节推动较强的作品开始，逐渐转向短篇隐喻、制度讽刺和复杂家族网络。最后的《红楼梦》不是简单最高点，而是检验读者能否同时处理人物声音、日常细节、诗歌和结构暗示。

## 使用方法

长篇可先读完整主线和关键回目，再补支线；每部记录人物怎样进入群体、叙述者怎样评价人物、重复关目如何产生变化。读《聊斋志异》时按主题选篇，同时比较短篇与章回小说的节奏。
