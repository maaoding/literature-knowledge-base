# 明清小说入门

明清小说可以按五种入口阅读：历史演义、英雄传奇、神魔小说、讽刺小说和世情小说。

## 阅读顺序

1. [《三国演义》](/works/三国演义)：历史演义和谋略叙事。
2. [《水浒传》](/works/水浒传)：群像、江湖伦理和反抗想象。
3. [《西游记》](/works/西游记)：神魔想象和成长结构。
4. [《儒林外史》](/works/儒林外史)：讽刺士人和科举秩序。
5. [《红楼梦》](/works/红楼梦)：世情、家族和人物群像的高峰。

## 阅读提示

不要把明清小说只当故事库。它们分别发展出不同的叙事机制：战役推进、人物聚合、关卡结构、讽刺切片和日常细节。
