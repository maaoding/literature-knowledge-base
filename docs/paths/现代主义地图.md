# 现代主义地图

**目标：** 从象征、意识流、荒诞和元小说掌握二十世纪形式实验的主要方向。

## 阅读顺序

1. [恶之花](/works/恶之花)

   从现代都市和象征暗示进入前史。

2. [到灯塔去](/works/到灯塔去)

   观察视角、日常感知和时间流动。

3. [尤利西斯](/works/尤利西斯)

   进入语言、城市和神话结构的极限实验。

4. [等待戈多](/works/等待戈多)

   看战后戏剧如何削弱情节和意义保证。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
