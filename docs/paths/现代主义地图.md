# 现代主义地图

## 路径说明

这不是按年代压缩的现代主义史，而是一张形式问题地图：象征如何取代直陈，拼贴如何容纳文化碎片，意识流如何改变时间，荒诞戏剧如何撤去行动目标，元小说又如何暴露解释权。八部作品组成从前史到后现代延伸的挑战路线。

## 使用方法

建议分阶段完成，每读一部只抓住一种核心技法，并用一段话说明它改变了什么阅读习惯。遇到引文、典故和语言实验时先观察结构功能，再借作品页与文学史页补充背景，避免把阅读变成注释收集。

## 完成之后

可进入“拉美与后殖民文学”，考察这些形式如何处理殖民历史和国家叙事；也可转向“东方文学现代化”，比较非西方传统面对现代性时的选择与改写。
