# 现代小说入门

这条路径从短篇小说出发，经过现代主义寓言、心理现实主义和魔幻现实主义，观察现代小说如何处理现实、个体和历史。

## 阅读顺序

1. [《呐喊》](/works/呐喊)：以短篇进入现代中国的问题意识。
2. [《变形记》](/works/变形记)：用异化寓言感受现代主义的精神结构。
3. [《罪与罚》](/works/罪与罚)：观察心理、伦理和社会压力。
4. [《百年孤独》](/works/百年孤独)：理解历史记忆与魔幻现实主义。

## 适合谁

适合想快速建立现代小说阅读坐标的读者。

## 读完之后

可以向两个方向继续：想看社会结构，转向 [现实主义阅读](/paths/现实主义阅读)；想看形式实验，转向 [现代主义入门](/paths/现代主义入门)。
