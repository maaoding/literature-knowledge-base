# 推荐阅读

阅读路径按主题、难度和作品形态组织。这里的路径不是固定书单，而是给每类读者一个可靠入口：先从结构清楚的作品或文学史页面进入，再逐步转向复杂长篇和跨区域专题。世界文学路径现在覆盖古代史诗、中古故事、人文主义、启蒙、浪漫主义、现实主义、现代主义、拉美、亚非与移民写作。

## 路径原则

- 先建立问题意识，再追求作品数量。
- 同一主题下混合中国文学和世界文学，方便比较。
- 难度按“入门、进阶、挑战”递进，不按文学地位排序。
