# 东方文学现代化

**目标：** 从日本、印度和亚洲现代文学观察传统不是被抛弃，而是被重新组织。

## 阅读顺序

1. [源氏物语](/works/源氏物语)

   先建立物哀、宫廷与长篇心理叙事的传统坐标。

2. [世纪之交东方文学](/history/世纪之交东方文学)

   理解日本和印度的近代转型。

3. [雪国](/works/雪国)

   看传统审美如何与现代疏离结合。

4. [亚非文学复兴](/history/亚非文学复兴)

   扩展到民族独立、语言与殖民遗产。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
