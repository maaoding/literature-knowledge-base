# 东方文学现代化

<ReadingPathGoal />

## 路径说明

“东方现代化”不是一个统一过程。这条路径先建立印度与日本古典文学的形式坐标，再进入中国乡土小说、日本现代主义和南亚后殖民小说。传统在现代作品中可能成为语言资源、审美记忆，也可能成为需要反省的社会秩序。

路径刻意避免只用西方流派衡量亚洲文学。比较重点是作品如何面对新的城市、国家、殖民经验和个人意识，以及古典形式在这一过程中被保留、改写或重新发明。

## 使用方法

先理解每部作品的本地文体和历史背景，再做跨区域比较。记录自然景物、时间感、叙述语言和国家经验的变化；不要把“传统”自动等同于保守，也不要把形式实验自动等同于西化。
