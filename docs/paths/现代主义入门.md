# 现代主义入门

## 路径说明

现代主义的难点常被误认为“情节不好懂”，实际变化发生在叙述资格、时间经验和语言组织上。本路径先用结构明确的短篇建立问题意识，再进入意识流、拼贴诗和荒诞戏剧，最后接触以记忆重建时间的长篇实验。

## 使用方法

第一次阅读先标出视角转换、重复意象、时间跳跃和无法兑现的行动，不急于寻找唯一解释。第二次再追问：传统叙事中哪个稳定条件被取消了，形式上的陌生感怎样对应人物的孤独与时代的断裂。

## 完成之后

“现代主义地图”会扩大到象征主义、元小说和后现代组合结构；“拉美与后殖民文学”则展示形式实验如何与殖民历史、国家记忆和混杂身份结合。
