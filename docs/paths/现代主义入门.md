# 现代主义入门

现代主义入门不必从最难的长篇开始。可以先从短篇和结构清楚的作品进入，理解现代文学为什么会转向异化、断裂、循环和形式实验。

## 阅读顺序

1. [《变形记》](/works/变形记)：以短篇寓言进入现代个体的异化处境。
2. [《呐喊》](/works/呐喊)：比较中国现代小说对个人觉醒和社会压力的处理。
3. [《百年孤独》](/works/百年孤独)：理解时间循环、历史记忆和魔幻现实主义。

## 适合谁

适合想理解“为什么现代文学不像传统故事那样讲故事”的读者。

## 阅读提示

读现代主义时，不要急着寻找情节答案。可以先问：叙事为什么断裂？人物为什么无法解释自己的处境？形式本身在表达什么？
