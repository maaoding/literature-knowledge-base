# 古代文明与史诗

<ReadingPathGoal />

## 路径说明

这条路径不把古代文学压缩成“英雄冒险”。前两部从死亡和人的限度开始，随后进入共同体历史、家族责任和战争伦理，最终比较英雄、王权与建国使命。宗教文献和史诗并置，是为了观察它们承担的社会功能差异。

作品产生于不同语言、仪式和传承环境，不能直接套用同一英雄模型。比较应从文本内部的秩序出发：谁有权行动，神如何介入，人怎样面对死亡，以及个人选择如何影响家族和共同体。

## 使用方法

长篇史诗可使用可靠节选，但应保留开端、核心冲突和结局。为每部记录“人的限度、共同体危机、神圣秩序”三个维度，再比较相同意象在不同文明中的意义，避免只做人物强弱排名。
