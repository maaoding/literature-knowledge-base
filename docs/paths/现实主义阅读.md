# 现实主义阅读

现实主义阅读的重点不是“写得像不像现实”，而是文学如何组织社会细节，使人物处境、制度压力和伦理选择变得可见。

## 阅读顺序

1. [《呐喊》](/works/呐喊)：看现实批判如何压缩进短篇小说。
2. [《死魂灵》](/works/死魂灵)：用怪诞交易理解官僚语言和制度现实。
3. [《包法利夫人》](/works/包法利夫人)：观察欲望、消费和日常幻觉。
4. [《米德尔马契》](/works/米德尔马契)：追踪婚姻、职业和理想组成的社会网络。
5. [《樱桃园》](/works/樱桃园)：从潜台词和庄园变迁观察旧阶层退场。
6. [《罪与罚》](/works/罪与罚)：把社会压力推进心理和伦理深处。
7. [《安娜·卡列尼娜》](/works/安娜卡列尼娜)：把婚姻、欲望和社会判断放在一起读。
8. [《战争与和平》](/works/战争与和平)：把个人选择放入历史运动和家庭生活。

## 适合谁

适合想理解小说如何处理社会问题、心理冲突和历史背景的读者。

## 阅读提示

现实主义作品可以边读边记录三个层面：人物想要什么、社会允许什么、叙述者如何判断。
