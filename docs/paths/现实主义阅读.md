# 现实主义阅读

## 路径说明

现实主义并不只是把生活写得逼真。它把金钱、职业、婚姻、阶层和制度压力组织进人物的选择，使私人命运成为观察社会结构的入口。本路径从较清楚的城市成长故事出发，逐步进入全景社会、叙述距离与戏剧潜台词。

## 使用方法

为每部作品做三栏笔记：人物想要什么，社会允许什么，叙述者如何引导判断。不要只提炼“批判了什么”，还要观察场景调度、人物网络、语体反差和叙述视角如何把判断变成阅读经验。

## 完成之后

完成后可进入“战争与历史创伤”，看社会描写怎样承受战争与记忆的压力；也可转向“现代主义入门”，比较现代主义如何打断现实主义的连续叙事和稳定观察位置。
