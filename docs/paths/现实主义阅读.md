# 现实主义阅读

现实主义阅读的重点不是“写得像不像现实”，而是文学如何组织社会细节，使人物处境、制度压力和伦理选择变得可见。

## 阅读顺序

1. [《呐喊》](/works/呐喊)：看现实批判如何压缩进短篇小说。
2. [《傲慢与偏见》](/works/傲慢与偏见)：从日常礼仪和婚姻选择观察阶层结构。
3. [《罪与罚》](/works/罪与罚)：把社会压力推进心理和伦理深处。
4. [《战争与和平》](/works/战争与和平)：把个人选择放入历史运动和家庭生活。

## 适合谁

适合想理解小说如何处理社会问题、心理冲突和历史背景的读者。

## 阅读提示

现实主义作品可以边读边记录三个层面：人物想要什么、社会允许什么、叙述者如何判断。
