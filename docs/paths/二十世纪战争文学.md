# 二十世纪战争文学

**目标：** 从士兵经验、家族记忆与反乌托邦理解战争如何改变现实主义。

## 阅读顺序

1. [现实主义文学的深化](/history/现实主义文学深化)

   先建立战争、革命与工业文明的历史背景。

2. [老人与海](/works/老人与海)

   从行动和失败理解节制的英雄伦理。

3. [喧哗与骚动](/works/喧哗与骚动)

   观察历史创伤如何进入破碎记忆。

4. [现代主义文学的兴起](/history/现代主义文学兴起)

   比较战争如何同时推动形式实验。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
