# 战争与历史创伤

## 路径说明

本路径由原“二十世纪战争文学”扩展而来，仍保留旧名称以便搜索，但阅读范围前推至古代史诗。这样可以看见战争叙事如何从英雄荣誉和共同体信念，转向家庭生活、平民感受、破碎记忆与国家历史的争夺。

## 使用方法

每部作品都追问三件事：谁获得发言权，暴力被赋予何种正当性，战争结束后记忆怎样继续作用。比较史诗、历史小说和现代主义叙事时，重点观察视角如何从英雄中心移向家庭、失败者和不可靠的记忆。

## 完成之后

“现实主义阅读”可补足社会结构与日常生活的表现方法；“拉美与后殖民文学”则把创伤问题延伸到殖民遗产、国家分裂、流亡身份和历史书写权。
