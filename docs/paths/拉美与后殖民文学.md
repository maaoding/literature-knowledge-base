# 拉美与后殖民文学

**目标：** 理解殖民历史、神话时间、语言混杂和移民身份怎样改造现代小说。

## 阅读顺序

1. [虚构集](/works/虚构集)

   用迷宫和假书重写知识与现实。

2. [百年孤独](/works/百年孤独)

   把家族循环放入殖民历史与政治暴力。

3. [午夜之子](/works/午夜之子)

   让个人身体与国家独立同步又互相冲突。

4. [流亡与移民文学](/history/流亡与移民文学)

   把作品放回跨语写作和身份重构的历史。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
