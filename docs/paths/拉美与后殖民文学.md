# 拉美与后殖民文学

## 路径说明

路径从经典殖民冒险叙事起步，再转向拉美对现实边界和历史时间的改写，继而进入独立国家、移民经验与流亡语言。重点不是把不同地区归为同一种经验，而是比较谁有权讲述土地、国家和身份。

## 使用方法

阅读时分别标记地图与土地、命名与语言、家族与国家三组关系。注意叙述者是否可靠，奇异事件为何被当作日常，以及作品怎样借形式实验反驳单线历史。不要把魔幻现实主义当作所有后殖民文学的共同标签。

## 完成之后

可用“现代主义地图”追溯元小说和时间实验的形式来源，也可进入“东方文学现代化”，比较亚洲传统在殖民、民族国家和现代审美压力下形成的不同回应。
