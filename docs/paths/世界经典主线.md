# 世界经典主线

这条路径用少量高密度作品搭起世界文学的骨架：史诗源头、宗教长诗、现代小说自觉、现实主义和现代长篇。

## 阅读顺序

1. [《伊利亚特》](/works/伊利亚特)：战争、荣誉和英雄愤怒。
2. [《奥德赛》](/works/奥德赛)：归乡、冒险和身份确认。
3. [《神曲》](/works/神曲)：伦理旅程和宗教想象。
4. [《堂吉诃德》](/works/堂吉诃德)：现代小说的自我反思。
5. [《包法利夫人》](/works/包法利夫人)：现实主义的欲望与日常。
6. [《百年孤独》](/works/百年孤独)：历史记忆和现代长篇结构。

## 阅读提示

这不是最轻松的路线。适合每读完一部就回到文学史页，确认它解决了什么文体问题。
