# 世界经典主线

<ReadingPathGoal />

## 路径说明

所谓“主线”不是价值排名，而是七个能解释文学形式变化的节点。路径从英雄史诗转向归乡叙事和宗教旅程，再进入小说的自我反思、现代诗的公共声音、现实主义的社会细节，最后抵达循环历史与地方经验。

这条路径适合已经完成世界文学入门、希望把零散名著放回历史关系的读者。每部作品都只是一个入口，目标是掌握前后形式为什么发生变化，而不是把七部作品当成世界文学的全部。

## 使用方法

为每个阶段画一张简短关系图，记录主人公类型、时间结构、叙述声音和共同体想象。完成一部后回看上一部，写下新作品继承了什么、反对了什么，以及哪些地区和声音仍未进入这条主线。
