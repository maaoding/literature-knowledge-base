# 女性与婚姻叙事

这条路径从婚姻选择、财产秩序和家族结构理解小说中的女性处境。它适合把“爱情故事”重新读成社会结构和主体选择的问题。

## 阅读顺序

1. [《傲慢与偏见》](/works/傲慢与偏见)：从婚姻市场观察阶层、判断力和女性主体。
2. [《红楼梦》](/works/红楼梦)：从家族结构和才情世界观察女性命运。
3. [《战争与和平》](/works/战争与和平)：把家庭、战争和精神成长连起来看。

## 适合谁

适合喜欢人物关系、家庭叙事和社会观察的读者。

## 阅读提示

注意婚姻背后的资源分配：财产、继承、教育、礼法和名誉，经常比感情本身更能解释人物处境。
