# 文艺复兴到启蒙

**目标：** 沿着人文主义、宗教冲突和公共理性理解近代欧洲文学的形成。

## 阅读顺序

1. [十日谈](/works/十日谈)

   从市民故事和世俗判断开始。

2. [堂吉诃德](/works/堂吉诃德)

   看小说如何反思阅读与现实。

3. [失乐园](/works/失乐园)

   进入自由意志、堕落和革命时代。

4. [格列佛游记](/works/格列佛游记)

   用幻想旅行检验理性与制度。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
