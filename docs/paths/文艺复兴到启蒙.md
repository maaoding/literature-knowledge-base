# 文艺复兴到启蒙

<ReadingPathGoal />

## 路径说明

这条路径追踪近代欧洲文学怎样重新想象人。市民故事和狂欢身体扩大世俗经验，现代小说反思旧文本，戏剧与宗教史诗处理规范和自由意志，启蒙小说则把个人劳动、旅行观察与帝国扩张放入同一世界。

路径不是从黑暗走向理性的进步叙事。每一次个人解放都伴随新的边界：欲望会伤害他人，理性可能产生傲慢，自主劳动也可能建立在占有和殖民关系上。文学的重要作用正是让这些矛盾同时可见。

## 使用方法

记录每部作品如何定义“自由的人”，并追问这种自由需要哪些社会条件、排除了谁。戏剧和故事集可以先读代表篇目，长篇则保持完整主线；完成阶段后比较讽刺对象和叙述者自身是否也受到反讽。
