# 浪漫主义到现实主义

**目标：** 比较想象与社会观察，理解十九世纪文学并非简单的流派替代。

## 阅读顺序

1. [浮士德](/works/浮士德)

   理解现代人的无限追求。

2. [巴黎圣母院](/works/巴黎圣母院)

   进入浪漫主义的历史与边缘人物。

3. [白鲸](/works/白鲸)

   看浪漫象征、现实知识和现代执念如何进入同一部小说。

4. [包法利夫人](/works/包法利夫人)

   观察浪漫幻想在现实生活中的后果。

5. [玩偶之家](/works/玩偶之家)

   把家庭冲突推进法律和主体自由。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
