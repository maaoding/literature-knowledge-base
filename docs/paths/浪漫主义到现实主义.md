# 浪漫主义到现实主义

<ReadingPathGoal />

## 路径说明

浪漫主义和现实主义并非想象与现实的简单对立。这条路径从强烈主观情感与无限追求开始，经过历史小说、诗体小说和象征性执念，最后观察现实主义如何吸收并反省浪漫愿望，把它放回金钱、婚姻和法律关系。

作品顺序强调连续性：现实主义人物仍然拥有理想和欲望，只是叙事更仔细地追踪它们需要的社会条件与实际代价。戏剧放在最后，是为了看长篇社会分析如何被压缩成一次公开行动。

## 使用方法

每部记录人物想象中的生活和实际拥有的资源，再观察叙述者是否支持这种想象。比较自然、城市、建筑和家庭空间的功能；不要只根据作品年代给流派贴标签，而要用语言、视角和结构说明判断。
