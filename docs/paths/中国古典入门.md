# 中国古典入门

这条路径从诗歌源头进入，再经过历史叙事，最后转向古典长篇小说。它的目标不是一次读完中国古典文学，而是建立三种基本能力：读抒情、读人物、读结构。

## 阅读顺序

1. [《诗经》](/works/诗经)：建立抒情和礼乐传统的起点。
2. [《楚辞》](/works/楚辞)：补上强烈个人抒情、神话想象和政治人格。
3. [《史记》](/works/史记)：理解人物传记和历史判断的叙事传统。
4. [《红楼梦》](/works/红楼梦)：进入古典小说的结构和人物群像。

## 适合谁

适合想从中国文学史起点建立整体感的读者。

## 读完之后

如果你更喜欢诗歌，可以转向 [诗歌传统入门](/paths/诗歌传统入门)；如果你更喜欢小说结构，可以继续读 [明清小说](/history/明清小说) 和 [曹雪芹](/authors/曹雪芹)。
