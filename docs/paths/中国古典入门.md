# 中国古典入门

<ReadingPathGoal />

## 路径说明

这条路径不按朝代平均分配篇幅，而选择六个能够改变阅读方式的节点。前四部先建立诗歌和思想语言，《史记》把判断转入人物与历史，《红楼梦》则让此前的抒情、伦理和叙事资源在长篇小说中重新汇合。

阅读重点不是记忆文学常识，而是辨认不同文体怎样发言：诗歌依靠声音和意象，语录与寓言依靠对话和思想转折，历史叙事通过取舍塑造人物，小说则把多种声音放进持续变化的关系网络。

## 使用方法

每完成一个阶段，选一段最有感触的文字，记录说话者、对象、语气和判断依据。遇到注释先解决妨碍理解的部分，不必在第一次阅读时穷尽典故；先保持整体声音，再回看细节。
