# 诗歌传统入门

这条路径从集体歌唱、个人抒情到士人心境，帮助你建立中国诗歌阅读的基本坐标。

## 阅读顺序

1. [《诗经》](/works/诗经)：理解风、雅、颂和比兴传统。
2. [《楚辞》](/works/楚辞)：进入浪漫想象、政治人格和个人抒情。
3. [李白](/authors/李白)：感受盛唐诗歌的自由气势和想象力。
4. [杜甫](/authors/杜甫)：理解诗史意识和现实关怀。
5. [苏轼](/authors/苏轼)：观察宋代士人的旷达、反省和人生境界。
6. [《漱玉词》](/works/漱玉词)：比较日常意象在闲愁与离乱经验中的不同重量。

## 适合谁

适合觉得古诗难入门，但想建立一条清楚阅读线索的读者。

## 阅读提示

先读情绪和意象，再读典故和格律。不要一开始就把诗歌阅读变成注释阅读。
