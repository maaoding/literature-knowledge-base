# 诗歌传统入门

<ReadingPathGoal />

## 路径说明

这条路径把不同语言传统放在“谁在说话、怎样组织声音”这一共同问题下。它从仪式和共同体歌唱开始，经过鲜明个人抒情、短诗与词体，最后进入自由诗和现代都市。作品之间不是影响链，而是可比较的声音坐标。

跨传统阅读时应尊重形式差异。《诗经》的章法、鲁拜的四行结构、和歌与词的节奏都不能直接互换；真正可比较的是它们如何压缩经验、安排说话者，并让私人情感连接社会和时间。

## 使用方法

先朗读，再查注释。每部选择三至五首代表作，记录重复、停顿、意象和说话对象；第二遍才处理典故与历史语境。译诗部分可对照两种中文译本，关注语气差异而非追求唯一答案。
