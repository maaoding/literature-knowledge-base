# 中古世界的信仰与故事

**目标：** 在欧洲、阿拉伯、波斯与日本之间比较信仰、宫廷和讲故事传统。

## 阅读顺序

1. [贝奥武甫](/works/贝奥武甫)

   进入英雄伦理与死亡意识。

2. [一千零一夜](/works/一千零一夜)

   理解讲述如何延缓死亡并容纳多文化故事。

3. [列王纪](/works/列王纪)

   从民族史诗进入波斯历史记忆。

4. [源氏物语](/works/源氏物语)

   用宫廷关系和物哀感受另一种时间。

## 使用方法

不必一次读完所有作品。每完成一步，回到相关文学史页面确认时代、文体和前后关系，再进入下一部作品。
