# 史诗与神话入门

## 路径说明

这条路径不按文明逐一罗列神话，而是追踪几种反复出现的叙事动力：英雄面对死亡，远行者寻找归宿，人物在变形与试炼中重新确认身份。作品跨越两河、希腊、印度、中国与阿拉伯传统，适合建立比较阅读的第一组坐标。

## 使用方法

每读一部，记录英雄为何出发、谁规定秩序、故事怎样解释灾难与死亡。遇到神谱和支线时不必追求记全，优先辨认母题如何改变，以及口传痕迹、框架结构和书面改写分别带来什么效果。

## 完成之后

完成后可沿“古代文明与史诗”补足文明背景，或进入“中古世界的信仰与故事”，观察神话叙事怎样转向宗教秩序、骑士伦理和跨地域故事网络。
