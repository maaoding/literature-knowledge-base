# 史诗与神话入门

这条路径从史诗、神魔小说和民间故事进入文学想象的源头。

## 阅读顺序

1. [《伊利亚特》](/works/伊利亚特)：英雄、战争和荣誉。
2. [《奥德赛》](/works/奥德赛)：归乡、诱惑和冒险。
3. [《西游记》](/works/西游记)：神魔世界和修行结构。
4. [《一千零一夜》](/works/一千零一夜)：框架叙事和故事延续。

## 阅读提示

神话和史诗不要只读情节。重点看它们如何解释共同体的价值、恐惧和愿望。
