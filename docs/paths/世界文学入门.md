# 世界文学入门

## 路径说明

这不是按国别罗列的名著清单，而是一条从“故事为什么要继续讲”出发的文体路径。故事集建立叙述欲望，悲剧训练对行动和责任的判断，社会小说要求辨认叙述反讽，现代小说则改变现实与时间的表现方式。

六部作品横跨多个传统，但比较时不追求把差异抹平。应先理解作品内部的社会和形式规则，再观察相似问题如何获得不同答案：人怎样面对命运、制度、亲密关系和无法解释的历史。

## 使用方法

每部作品只选择一个核心问题持续追踪，并写下它与上一部的一个相似点和一个差异。遇到篇幅较长的作品可采用可靠节选或分段阅读，但要保留开头、关键转折和结尾的完整联系。
