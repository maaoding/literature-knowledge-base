# 世界文学入门

这条路径按戏剧、小说和长诗三条线进入世界文学经典。它不按年代严格推进，而是优先选择能代表关键文体和问题意识的作品。

## 阅读顺序

1. [《哈姆雷特》](/works/哈姆雷特)：理解戏剧冲突和人性困境。
2. [《堂吉诃德》](/works/堂吉诃德)：进入现代小说的自反传统。
3. [《傲慢与偏见》](/works/傲慢与偏见)：在日常叙事中观察阶层和婚姻。
4. [《神曲》](/works/神曲)：理解伦理旅程和欧洲文学想象。

## 适合谁

适合已经读过少量名著，希望建立世界文学结构感的读者。

## 读完之后

如果你对社会现实更感兴趣，接 [现实主义阅读](/paths/现实主义阅读)；如果你对现代文学的荒诞和断裂更感兴趣，接 [现代主义入门](/paths/现代主义入门)。
